# 单位半群（Monoid）

这个相对简单，就是一个代数结构。当集合SSS 上存在一个二元运算∗:S×S→S*:S\times S\rightarrow S∗:S×S→S 和一个单位元e∈Se\in Se∈S 满足：

  1. ∀a∈S,  e∗a=a∗e=a\forall a\in S,\;e*a=a*e=a∀a∈S,e∗a=a∗e=a.

  2. ∀a,b,c∈S,  a∗(b∗c)=(a∗b)∗c\forall a,b,c\in S,\; a*(b*c)=(a*b)*c∀a,b,c∈S,a∗(b∗c)=(a∗b)∗c.

那我们就称集合SSS 和这个运算、单位元形成了一个单位半群（幺半群，Monoid）

例：自态射形成了一个 Monoid。考虑一个 category C\mathcal{C}C 中只有一个元素aaa 和很多 morphism，那么这些 morphism 都是从aaa 到aaa 的箭头，他们形成了一个 Monoid。其中SSS 就是这些 morphism，二元运算是 morphism 的组合，单位元是 identity morphism。由于 category 的性质，故满足结合律和单位律，故形成了一个 Monoid。

# Monoidal Category

A monoidal category is a category C\mathcal{C}C quipped with a monoidal structure:

  • a bifunctor ⊗:C×C→C\otimes:\mathcal{C}\times\mathcal{C}\rightarrow\mathcal{C}⊗:C×C→C.

  • an object III called the monoid unit.

  • three natural isomorphisms:

    1. for any three arguments A,B,CA,B,CA,B,C, there is a isomorphism α\alphaα:

      αA,B,C:A⊗(B⊗C)≅(A⊗B)⊗C\alpha_{A,B,C}:A\otimes (B\otimes C)\cong (A\otimes B)\otimes C αA,B,C​:A⊗(B⊗C)≅(A⊗B)⊗C

    2. for any argument AAA, there are two isomorphisms:

      λA:I⊗A≅AρA:A⊗I≅A\lambda_A:I\otimes A\cong A\\ \rho_A:A\otimes I\cong A λA​:I⊗A≅AρA​:A⊗I≅A

  • the three natural isomorphisms should satisfy the coherence conditions:

    1. for all A,B,C,D∈CA,B,C,D\in\mathcal{C}A,B,C,D∈C, the following diagram commutes:

    2. for all A,B∈CA,B\in\mathcal{C}A,B∈C, the following diagram commutes:

关于这个定义，可以说明几点。

  1. 这个 monoidal structure 里的元素（譬如单位元），都是 category C\mathcal{C}C 里面的 object。也就是说我们希望这个 category 本身就有一点 monoid 的结构。但是它不是一个 monoid 在于，结合律和单位律都是同构而不是相等。譬如在 category Set\textbf{Set}Set 中，就是集合的同构而不是集合的相等。{(a,(b,c))}\{(a,(b,c))\}{(a,(b,c))} 和{((a,b),c)}\{((a,b),c)\}{((a,b),c)} 就是同构但不相等的。

  2. 其次下面的 coherence conditions 其实是，我们可以利用α\alphaα 找出很多个诸如下面这种同构：

    (((A1⊗A2)⊗A3)⊗...⊗An)≅(A1⊗(A1⊗(A3⊗...⊗An)))(((A_1\otimes A_2)\otimes A_3)\otimes ...\otimes A_n)\cong(A_1\otimes (A_1\otimes (A_3\otimes ...\otimes A_n))) (((A1​⊗A2​)⊗A3​)⊗...⊗An​)≅(A1​⊗(A1​⊗(A3​⊗...⊗An​)))

    因为你可以选择拆括号时不同的顺序，从而选择α\alphaα 的不同的 arguments。coherence condition 就说明，这些同构都应该是同一个。

例：把 category Set\textbf{Set}Set 补充成一个 monoidal category。bifunctor 选择笛卡尔积，即A⊗B={(a,b)∣a∈A,b∈B}A\otimes B=\{(a,b)\mid a\in A,b\in B\}A⊗B={(a,b)∣a∈A,b∈B}。然后 unit object 选择为一个特殊的单元素集{∗}\{*\}{∗}。可以验证譬如{((1,2),3),((4,5),6)}≅{(1,(2,3)),(4,(5,6))}\{((1,2),3),((4,5),6)\}\cong \{(1,(2,3)),(4,(5,6))\}{((1,2),3),((4,5),6)}≅{(1,(2,3)),(4,(5,6))}，以及{(1,∗),(2,∗)}≅{1,2}\{(1,*),(2,*)\}\cong\{1,2\}{(1,∗),(2,∗)}≅{1,2} 这样有自然的 isomorphism 满足要求和 coherence condition。

# Monoid in monoidal category

下面我们定义，在一个 monoidal category 上自然导出一个 monoid：

A monoid (M,μ,η)(M,\mu,\eta)(M,μ,η) in a monoidal category (C,⊗,I)(\mathcal{C},\otimes,I)(C,⊗,I) is an object MMM together with two morphism μ,η\mu,\etaμ,η:

  • μ:M⊗M→M\mu:M\otimes M\rightarrow Mμ:M⊗M→M called multiplication.
  • η:I→M\eta:I\rightarrow Mη:I→M called unit.

例：考虑Set\textbf{Set}Set 是一个 monoidal category，那么我们选择其中一个 object，譬如M={0,1}M=\{0,1\}M={0,1}。那么M⊗M={(0,0),(0,1),(1,0),(1,1)}M\otimes M=\{(0,0),(0,1),(1,0),(1,1)\}M⊗M={(0,0),(0,1),(1,0),(1,1)}。我们可以找到一个 morphism μ\muμ，即模 2 乘法：

μ(0,0)=0μ(0,1)=0μ(1,0)=0μ(1,1)=1\mu(0,0)=0\\ \mu(0,1)=0\\ \mu(1,0)=0\\ \mu(1,1)=1 μ(0,0)=0μ(0,1)=0μ(1,0)=0μ(1,1)=1

对于η\etaη，我们需要找到一个{∗}→{0,1}\{*\}\rightarrow\{0,1\}{∗}→{0,1} 的映射，很显然有两个选择。如果我们选择η∗=0\eta_*=0η∗​=0 会怎样？那么第二个 diagram 将不 commute！因为I⊗M⟶η⊗1M⊗M⟶μMI\otimes M\stackrel{\eta\otimes 1}{\longrightarrow}M\otimes M\stackrel{\mu}{\longrightarrow}MI⊗M⟶η⊗1​M⊗M⟶μ​M 就会有映射：(∗,1)↦(0,1)↦0(*,1)\mapsto(0,1)\mapsto 0(∗,1)↦(0,1)↦0，显然这和直接用消去映射λ(∗,1)=1\lambda(*,1)=1λ(∗,1)=1 得到的结果不一样。

所以我们需要选择η∗=1\eta_*=1η∗​=1，这样就满足了要求。这个例子也阐释了，我们是怎么利用已有的 monoidal category 中那些 monoid structure（即α,λ,ρ\alpha,\lambda,\rhoα,λ,ρ）构造一个 monoid 的。

# Monad

最终我们定义一个特殊情况：Monad is a monoid in the (monoidal) category of endofunctors.

考虑一个 category C\mathcal{C}C，显然可以找到一个 category D\mathcal{D}D：

  • D\mathcal{D}D 中的 object 是C→C\mathcal{C}\rightarrow\mathcal{C}C→C 的 functor。
  • D\mathcal{D}D 中的 morphism 是 functor 之间的 natural transformation。

那么D\mathcal{D}D 是一个 monoidal category：

  • 对于两个 functor F,G∈DF,G\in\mathcal{D}F,G∈D，F⊗G=F∘G∈DF\otimes G=F\circ G\in\mathcal{D}F⊗G=F∘G∈D。
  • I=1CI=1_{\mathcal{C}}I=1C​ 即C\mathcal{C}C 上的 identity functor。

不难验证，这满足 monoidal category 的要求。实际上，D\mathcal{D}D 是一个 strict monoidal category，因为根据 category 和 functor 的要求，实际上就有F∘(G∘H)=(F∘G)∘HF\circ(G\circ H)=(F\circ G)\circ HF∘(G∘H)=(F∘G)∘H，即不是同构而是直接相等。

那么我们就可以找出D\mathcal{D}D 中的一个元素和两个 morphism，然后导出一个 monoid，即是 monad：

A Monad on C\mathcal{C}C consists of an endofunctor T:C→CT:\mathcal{C}\rightarrow\mathcal{C}T:C→C together with two natural transformations η:1C→T\eta:1_{\mathcal{C}}\rightarrow Tη:1C​→T and μ:T∘T→T\mu:T\circ T\rightarrow Tμ:T∘T→T.

例：我们仍考虑 category Set\textbf{Set}Set，那么 powerset 实际上就是一个 Monad。考虑一个 endofunctor T(A∈Set)=2A∈SetT(A\in\textbf{Set})=2^A\in\textbf{Set}T(A∈Set)=2A∈Set，那么我们可以定义ηA:1Set(A)→T(A)\eta_A:1_{\textbf{Set}}(A)\rightarrow T(A)ηA​:1Set​(A)→T(A) 为ηA(a∈A)={a}\eta_A(a\in A)=\{a\}ηA​(a∈A)={a}，譬如对于A={1,2,3}A=\{1,2,3\}A={1,2,3}，那么A↦ηA{{1},{2},{3}}A\stackrel{\eta_A}{\mapsto}\{\{1\},\{2\},\{3\}\}A↦ηA​​{{1},{2},{3}}。

然后定义μA:T(T(A))→T(A)\mu_A:T(T(A))\rightarrow T(A)μA​:T(T(A))→T(A) 其实就是把集合 flatten 一层。譬如对于A={{{1,2},{3,4}},{{5}}}A=\{\{\{1,2\},\{3,4\}\},\{\{5\}\}\}A={{{1,2},{3,4}},{{5}}}，那么A↦ηA{{1,2,3,4},{5}}A\stackrel{\eta_A}{\mapsto}\{\{1,2,3,4\},\{5\}\}A↦ηA​​{{1,2,3,4},{5}}。这就形成了一个 Monad。

注意，实际上 functor TTT 应该不仅描述了 object 间的映射关系，还需要说明 morphism 间的映射关系。而Set\textbf{Set}Set 上的 morphism 是集合之间的函数，那么对于 powerset 这个TTT，一个很自然的映射就是把函数（譬如f(x)=x+1f(x)=x+1f(x)=x+1）apply 在 powerset 里的每个 set。
