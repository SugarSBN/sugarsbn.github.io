# Lattice

Let (S,⊑)(S,\sqsubseteq)(S,⊑) be a partial order.

(S,⊑)(S,\sqsubseteq)(S,⊑) is a lattice if the meet and join of any pair of elements of SSS always exists.

(S,⊑)(S,\sqsubseteq)(S,⊑) is a complete lattice if the meet and join of any subset of elements of SSS always exists.

# Galois connection

Let (C,⊆)(C,\subseteq)(C,⊆) and (A,⊑)(A,\sqsubseteq)(A,⊑) be complete lattice, and let α:C→A,γ:A→C\alpha:C\rightarrow A,\gamma:A\rightarrow Cα:C→A,γ:A→C.

We say (C,A,α,γ)(C,A,\alpha,\gamma)(C,A,α,γ) is a (monotone) Galois connection if for all c∈C,a∈Ac\in C,a\in Ac∈C,a∈A:

α(c)⊑a  ⟺  c⊆γ(a)\alpha(c)\sqsubseteq a\iff c\subseteq \gamma(a) α(c)⊑a⟺c⊆γ(a)

Lemma: In Galois connection, α,γ\alpha,\gammaα,γ are monotonic.

Proof:

∵∀c∈C.  α(c)⊑α(c)∴∀c∈C.  c⊆γ(α(c))∴∀c′⊆c.  c′⊆γ(α(c))∴∀c′⊆c.  α(c′)⊑α(c)\begin{aligned} & \because \forall c\in C.\;\alpha(c)\sqsubseteq\alpha(c)\\ & \therefore\forall c\in C.\;c\subseteq\gamma(\alpha(c))\\ & \therefore \forall c'\subseteq c.\;c'\subseteq\gamma(\alpha(c))\\ & \therefore \forall c'\subseteq c.\;\alpha(c')\sqsubseteq\alpha(c) \end{aligned} ​∵∀c∈C.α(c)⊑α(c)∴∀c∈C.c⊆γ(α(c))∴∀c′⊆c.c′⊆γ(α(c))∴∀c′⊆c.α(c′)⊑α(c)​

Similarly we can prove that γ\gammaγ is monotonic.

□\square□.

Equivalent Definition: (C,A,α,γ)(C,A,\alpha,\gamma)(C,A,α,γ) is a (monotone) Galois connection if for all c∈C,a∈Ac\in C,a\in Ac∈C,a∈A:

c⊆γ(α(c)) and α(γ(a))⊑ac\subseteq \gamma(\alpha(c))\text{ and }\alpha(\gamma(a))\sqsubseteq a c⊆γ(α(c)) and α(γ(a))⊑a

Proof:

  • (=>) Assume α(c)⊑a  ⟺  c⊆γ(a)\alpha(c)\sqsubseteq a\iff c\subseteq\gamma(a)α(c)⊑a⟺c⊆γ(a).

    Because α(c)⊑α(c)\alpha(c)\sqsubseteq\alpha(c)α(c)⊑α(c), therefore c⊆γ(α(c))c\subseteq\gamma(\alpha(c))c⊆γ(α(c)). Similarly it ca be proved that α(γ(a))⊑a\alpha(\gamma(a))\sqsubseteq aα(γ(a))⊑a because γ(a)⊆γ(a)\gamma(a)\subseteq\gamma(a)γ(a)⊆γ(a).

  • (<=) Assume c⊆γ(α(c))c\subseteq\gamma(\alpha(c))c⊆γ(α(c)) and α(γ(a))⊑a\alpha(\gamma(a))\sqsubseteq aα(γ(a))⊑a.

    If α(c)⊑a\alpha(c)\sqsubseteq aα(c)⊑a, then since γ\gammaγ is monotonic:

    c⊆γ(α(c))⊑γ(a)c\subseteq\gamma(\alpha(c))\sqsubseteq \gamma(a) c⊆γ(α(c))⊑γ(a)

    Similarly, we can prove that if c⊆γ(a)c\subseteq\gamma(a)c⊆γ(a), then α(c)⊑a\alpha(c)\sqsubseteq aα(c)⊑a…

□\square□.

Proof: By equivalent definition.

□\square□.

Lemma: α\alphaα preserves suprema and γ\gammaγ preserves infima. i.e., given a subset X⊆C,Y⊆AX\subseteq C,Y\subseteq AX⊆C,Y⊆A:

α(⋃x∈Xx)=⋁x∈Xα(x)γ(⋀y∈Yy)=⋂y∈Yγ(y)\alpha(\bigcup_{x\in X} x)=\bigvee_{x\in X}\alpha(x)\qquad \gamma(\bigwedge_{y\in Y}y)=\bigcap_{y\in Y}\gamma(y) α(x∈X⋃​x)=x∈X⋁​α(x)γ(y∈Y⋀​y)=y∈Y⋂​γ(y)

Proof: 我们只需要证明，α(⋃x)\alpha(\bigcup x)α(⋃x) 是{α(x)∣x∈X}\{\alpha(x)\mid x\in X\}{α(x)∣x∈X} 的最小公共上界。

因为∀x∈X.  x⊆⋃x\forall x\in X.\;x\subseteq\bigcup x∀x∈X.x⊆⋃x，那么根据单调性有∀x∈X.  α(x)⊑α(⋃x)\forall x\in X.\;\alpha(x)\sqsubseteq \alpha(\bigcup x)∀x∈X.α(x)⊑α(⋃x). 下面我们证明它是最小的公共上界。

假设还有一个公共上界yyy : ∀x∈X.  α(x)⊑y\forall x\in X.\;\alpha(x)\sqsubseteq y∀x∈X.α(x)⊑y, 根据 Galois connection 就有∀x∈X.  x⊆γ(y)\forall x\in X.\;x\subseteq\gamma(y)∀x∈X.x⊆γ(y). 因此⋃x∈Xx⊆γ(y)\bigcup_{x\in X}x\subseteq\gamma(y)⋃x∈X​x⊆γ(y) (γ(y)\gamma(y)γ(y) 是XXX 的一个公共上界，故大于其最小公共上界)。 那么就有α(⋃x)⊑α(γ(y))⊑y\alpha(\bigcup x)\sqsubseteq\alpha(\gamma(y))\sqsubseteq yα(⋃x)⊑α(γ(y))⊑y。因此，α(⋃x)\alpha(\bigcup x)α(⋃x) 是所有公共上界中最小的一个。下界同理。

Proof: Since γ∘α≥idC\gamma\circ\alpha\geq\text{id}_Cγ∘α≥idC​ and α∘γ≤idA\alpha\circ\gamma\leq\text{id}_Aα∘γ≤idA​:

α∘γ∘α(c)=α∘(γ∘α)(c)⊒α(c)α∘γ∘α(c)=(α∘γ)∘α(c)⊑α(c)\alpha\circ\gamma\circ\alpha(c)=\alpha\circ(\gamma\circ\alpha)(c)\sqsupseteq \alpha(c)\\ \alpha\circ\gamma\circ\alpha(c)=(\alpha\circ\gamma)\circ\alpha(c)\sqsubseteq \alpha(c) α∘γ∘α(c)=α∘(γ∘α)(c)⊒α(c)α∘γ∘α(c)=(α∘γ)∘α(c)⊑α(c)

thus α∘γ∘α=α\alpha\circ\gamma\circ\alpha=\alphaα∘γ∘α=α. And the other cas e is similar.

# Galois Embedding

Let (C,A,α,γ)(C,A,\alpha,\gamma)(C,A,α,γ) a Galois connection.

It forms a Galois embedding if α∘γ=idA\alpha\circ\gamma=\text{id}_Aα∘γ=idA​.

Lemma: Given a Galois connection (C,A,α,γ)(C,A,\alpha,\gamma)(C,A,α,γ), the following conditions are equivalent:

  1. It forms a Galois embedding.

  2. α\alphaα is a surjective function.

  3. γ\gammaγ is an injective function.

Proof:

  • (1 => 2) 若α∘γ=idA\alpha\circ\gamma=\text{id}_Aα∘γ=idA​，那么显然，∀a∈A.  ∃y=γ(a)\forall a\in A.\;\exists y=\gamma(a)∀a∈A.∃y=γ(a) 使得α(y)=a\alpha(y)=aα(y)=a。故α\alphaα 是满射。

  • (2 => 3) 若α\alphaα 是满射，反设存在a1≠a2∈Aa_1\neq a_2\in Aa1​=a2​∈A 使得γ(a1)=γ(a2)≜c\gamma(a_1)=\gamma(a_2)\triangleq cγ(a1​)=γ(a2​)≜c。因为α\alphaα 是满射，所以存在c1,c2∈Cc_1,c_2\in Cc1​,c2​∈C 使得α(c1)=a1,α(c2)=a2\alpha(c_1)=a_1,\alpha(c_2)=a_2α(c1​)=a1​,α(c2​)=a2​。所以有：

α(c)=α(γ(a1))⊑a1α(c)=α(γ(a2))⊑a2\alpha(c)=\alpha(\gamma(a_1))\sqsubseteq a_1\\ \alpha(c)=\alpha(\gamma(a_2))\sqsubseteq a_2 α(c)=α(γ(a1​))⊑a1​α(c)=α(γ(a2​))⊑a2​

​ 又因为c1⊆γ(α(c1))=γ(a1)=cc_1\subseteq \gamma(\alpha(c_1))=\gamma(a_1)=cc1​⊆γ(α(c1​))=γ(a1​)=c，同理也有c2⊆cc_2\subseteq cc2​⊆c，所以有：

a1=α(c1)⊑α(c)⊑a1a2=α(c2)⊑α(c)⊑a2a_1=\alpha(c_1)\sqsubseteq\alpha(c)\sqsubseteq a_1\\ a_2=\alpha(c_2)\sqsubseteq\alpha(c)\sqsubseteq a_2 a1​=α(c1​)⊑α(c)⊑a1​a2​=α(c2​)⊑α(c)⊑a2​

​ 因此可以夹逼出a1=α(c)=a2a_1=\alpha(c)=a_2a1​=α(c)=a2​，矛盾。

  • (3 => 1) 若γ\gammaγ 是单射，反设存在a∈Aa\in Aa∈A 使得α(γ(a))=a′≠a\alpha(\gamma(a))=a'\neq aα(γ(a))=a′=a。同时套上γ\gammaγ，有：

    γ(a′)=γ(α(γ(a)))=γ(a)\gamma(a')=\gamma(\alpha(\gamma(a)))=\gamma(a) γ(a′)=γ(α(γ(a)))=γ(a)

    这和γ\gammaγ 是单射矛盾。

□\square□ 证毕。
