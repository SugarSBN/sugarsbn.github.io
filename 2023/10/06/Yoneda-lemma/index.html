yX(Y←fZ)=C(Y,X)→λg.g∘fC(Z,X)yX\left (Y\xleftarrow{f}Z\right )=\mathbf{C}(Y,X)\xrightarrow{\lambda g.g\circ f}\mathbf{C}(Z,X) yX(Yf​Z)=C(Y,X)λg.g∘f​C(Z,X)

对于Z←gZ′∈CopZ\xleftarrow{g}Z'\in\mathbf{C}^{op}Zg​Z′∈Cop：

Yoneda Lemma: For each small category C\mathbf{C}C, each object X∈CX\in\mathbf{C}X∈C and each presheaf F∈SetCopF\in\textbf{Set}^{\mathbf{C}^{op}}F∈SetCop, there is a bijection of sets:

ηX,F:SetCop(yX,F)≅F(X)\eta_{X,F}:\textbf{Set}^{\mathbf{C}^{op}}(yX,F)\cong F(X) ηX,F​:SetCop(yX,F)≅F(X)

which is natural in both XXX and FFF.

注意，yX,F:Cop→SetyX,F:\mathbf{C}^{op}\rightarrow\textbf{Set}yX,F:Cop→Set 都是 functor，而SetCop(yX,F)\textbf{Set}^{\mathbf{C}^{op}}(yX,F)SetCop(yX,F) 就是这两个 functor 之间的 natural transformation 的集合。而这个集合同构于F(X)F(X)F(X)。

下面我们试图给出这个同构映射，证明该引理。

给定一个θ:yX⇒F\theta:yX\Rightarrow Fθ:yX⇒F，那么有θX:yX(X)→F(X)\theta_X:yX(X)\rightarrow F(X)θX​:yX(X)→F(X)，因为yX(X)=C(X,X)yX(X)=\mathbf{C}(X,X)yX(X)=C(X,X)，所以θX(idX)∈F(X)\theta_X(\text{id}_X)\in F(X)θX​(idX​)∈F(X)。

# 理解和解释

我们重新审视以下 Yoneda Lemma 的式子，更简便地、它可以写为：

Nat(Hom(−,X),F)≅F(X)\textbf{Nat}(\text{Hom}(-,X),F)\cong F(X) Nat(Hom(−,X),F)≅F(X)

其中，Nat(F,G)\textbf{Nat}(F,G)Nat(F,G) 表示从FFF 到GGG 的 natural transformation 的集合。

而Hom(−,X):Cop→Set\text{Hom}(-,X):\mathbf{C}^{op}\rightarrow \textbf{Set}Hom(−,X):Cop→Set 是一个 Functor，FFF 也是Cop→Set\mathbf{C}^{op}\rightarrow \textbf{Set}Cop→Set 的 Functor。其中

Hom(−,X)(Y←fZ)=Hom(Y,X)→f∗=λg.g∘fHom(Z,X)\text{Hom}(-,X)\left (Y\xleftarrow{f}Z\right )=\text{Hom}(Y,X)\xrightarrow{f^*=\lambda g.g\circ f}\text{Hom}(Z,X) Hom(−,X)(Yf​Z)=Hom(Y,X)f∗=λg.g∘f​Hom(Z,X)

为了理解这个抽象的同构关系，我们先看几个引理，试图感觉一下，它实例化后的含义。

Collaroy 1. 如果F=Hom(−,Y)F=\text{Hom}(-,Y)F=Hom(−,Y) 也是一个Cop→Set\mathbf{C}^{op}\rightarrow\textbf{Set}Cop→Set，那么代入可以直接得到：

Hom(X,Y)≅Nat(Hom(−,X),Hom(−,Y))\text{Hom}(X,Y)\cong\textbf{Nat}(\text{Hom}(-,X),\text{Hom}(-,Y)) Hom(X,Y)≅Nat(Hom(−,X),Hom(−,Y))

这说明什么？别忘记，Nat(−,−)\textbf{Nat}(-,-)Nat(−,−) 是 functor category SetCop\textbf{Set}^{\mathbf{C}^{op}}SetCop 里两个 object 之间的 morphism 集合！

实际上，我们找到了一个 embedding，把C\mathbf{C}C 嵌入到了SetCop\textbf{Set}^{\mathbf{C}^{op}}SetCop：

X↦Hom(−,X)Hom(X,Y)↦Nat(Hom(−,X),Hom(−,Y))\begin{aligned} X&\mapsto \text{Hom}(-,X)\\ \text{Hom}(X,Y)&\mapsto\textbf{Nat}(\text{Hom}(-,X),\text{Hom}(-,Y)) \end{aligned} XHom(X,Y)​↦Hom(−,X)↦Nat(Hom(−,X),Hom(−,Y))​

对于SetCop\textbf{Set}^{\mathbf{C}^{op}}SetCop 中的 object，那些形如Hom(−,X)\text{Hom}(-,X)Hom(−,X) 被称为 representable functor（即被XXX represent 了）

那么这个嵌入说明了一个事情：

对于任何 Representable Functor 之间的 natural transformation，都是 arise from 一个它们代表元之间的 morphism。

即对于任意 natural transformation Hom(−,X)⇒θHom(−,Y)\text{Hom}(-,X)\xRightarrow{\theta}\text{Hom}(-,Y)Hom(−,X)θ​Hom(−,Y)，都存在一个 morphism X→fYX\xrightarrow{f}YXf​Y 和θ\thetaθ 对应。

给定X→fYX\xrightarrow{f}YXf​Y 后，很自然可以诱导出一个 natural transformation Hom(−,X)⇒θHom(−,Y)\text{Hom}(-,X)\xRightarrow{\theta}\text{Hom}(-,Y)Hom(−,X)θ​Hom(−,Y)：

θZ=λg.f∘g\theta_Z=\lambda g.f\circ g θZ​=λg.f∘g

而这个引理说明，这个诱导是可逆的，对于任何一个 natural transformation 也只有唯一的 morphism。

Collaroy 2. X≅YX\cong YX≅Y if and only if Hom(−,X)≅Hom(−,Y)\text{Hom}(-,X)\cong\text{Hom}(-,Y)Hom(−,X)≅Hom(−,Y)。

这其实是因为，X↦Hom(−,X)X\mapsto\text{Hom}(-,X)X↦Hom(−,X) 是一个 fully faithful 的 functor。

一个 functor F:C→DF:\mathbf{C}\rightarrow\mathbf{D}F:C→D 被称为 fully faithful，当且仅当HomC(X,Y)≅HomD(F(X),F(Y))\text{Hom}_\mathbf{C}(X,Y)\cong\text{Hom}_\mathbf{D}(F(X),F(Y))HomC​(X,Y)≅HomD​(F(X),F(Y))。

然后对于一个 fully faithful functor，有：如果F(X)≅F(Y)F(X)\cong F(Y)F(X)≅F(Y)，那么X≅YX\cong YX≅Y。

这是一个很有哲学意义的引理。它说明了，object 的 property，实际上都是从它和周围 object 的 relation 中决定的。

即对于两个 object X,YX,YX,Y，如果我们从任意一个第三方 object ZZZ 去观察它们（即 morphism 看作是一种观察，关系…）那么X≅YX\cong YX≅Y 就当且仅当，Hom(Z,X)≅Hom(Z,Y)\text{Hom}(Z,X)\cong\text{Hom}(Z,Y)Hom(Z,X)≅Hom(Z,Y)。

严格来说：

如果对于任意ZZZ，Hom(Z,X)≅Hom(Z,Y)\text{Hom}(Z,X)\cong\text{Hom}(Z,Y)Hom(Z,X)≅Hom(Z,Y)，并且这个同构 natural in ZZZ，那么就有X≅YX\cong YX≅Y。

如果取个 dual category，其实Hom(X,Z)≅Hom(Y,Z)\text{Hom}(X,Z)\cong\text{Hom}(Y,Z)Hom(X,Z)≅Hom(Y,Z) 也可以得到X≅YX\cong YX≅Y。

注：考虑同构η:Hom(Z,X)≅Hom(Z,Y)\eta:\text{Hom}(Z,X)\cong\text{Hom}(Z,Y)η:Hom(Z,X)≅Hom(Z,Y)，那么η\etaη natural in ZZZ 的含义是，对于任意Z1←fZ2∈CZ_1\xleftarrow{f}Z_2\in\mathbf{C}Z1​f​Z2​∈C，有：

Hom(f,Y)∘ηZ1=ηZ2∘Hom(f,X)\text{Hom}(f,Y)\circ\eta_{Z_1}=\eta_{Z_2}\circ\text{Hom}(f,X) Hom(f,Y)∘ηZ1​​=ηZ2​​∘Hom(f,X)

就是一个交换图。（注意，Hom(−,X)\text{Hom}(-,X)Hom(−,X) 是Cop→Set\mathbf{C}^{op}\rightarrow\textbf{Set}Cop→Set 的 functor）

我们重新回到 Yoneda Lemma 的原始形式。给定一个 object XXX 和一个 Functor F:Cop→SetF:\mathbf{C}^{op}\rightarrow\textbf{Set}F:Cop→Set，以及引导出的 functor Hom(−,X):Cop→Set\text{Hom}(-,X):\mathbf{C}^{op}\rightarrow\textbf{Set}Hom(−,X):Cop→Set。

什么是好得到的呢？F(X)F(X)F(X) 显然是知道的，因为F,XF,XF,X 都给了。

但从Hom(−,X)\text{Hom}(-,X)Hom(−,X) 到FFF 的 natural transformation 却是难以得知的，而且可能存在特别多的情况。

然后 Yoneda Lemma 就说明了，natural transformation 都是由F(X)F(X)F(X) 中的元素诱导出来的。
