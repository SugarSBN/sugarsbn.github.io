# 隐含子群问题

隐含子群问题：给定一个群GGG，一个函数f:G→Somethingf:G\rightarrow Somethingf:G→Something，满足存在一个子群HHH，使得fff 在HHH 的不同陪集上互异，同一个陪集上相同。试求这个HHH。

  • 例 1：若给定群G=⟨{0,1}n,⊕⟩G=\langle \{0,1\}^n,\oplus\rangleG=⟨{0,1}n,⊕⟩，fff 为一个函数满足f(x)=f(x⊕s)f(x)=f(x\oplus s)f(x)=f(x⊕s)。那么如果有一个计算fff 的黑盒，我们想求sss，就相当于求子群K={0,s}K=\{0,s\}K={0,s}。
  • 例 2：离散对数问题。给定群G=⟨Zr×Zr,+(mod  r)⟩G=\langle \mathbb{Z}_r\times\mathbb{Z}_r,+(mod\;r)\rangleG=⟨Zr​×Zr​,+(modr)⟩，和函数fff 满足f(x1,x2)=ax1bx2f(x_1,x_2)=a^{x_1}b^{x_2}f(x1​,x2​)=ax1​bx2​。那么要求s=logabs=log_abs=loga​b 的话（模rrr 意义），那么就相当于求解一个子群：K=⟨{(l,−ls)  ∣  l∈Zr},+(mod  r)⟩K=\langle\{(l,-ls)\;|\;l\in\mathbb{Z}_r\},+(mod\;r)\rangleK=⟨{(l,−ls)∣l∈Zr​},+(modr)⟩，其实也就是这个周期是一个二维的。
