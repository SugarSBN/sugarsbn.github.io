Theorem (Kraus Theorem): A map NA→B\mathcal{N}_{A\rightarrow B}NA→B​ from a finite-dimensional Hilbert space HA\mathcal{H}_AHA​ to a finite-dimensional space HB\mathcal{H}_BHB​ is:

  • Convex linear:

    NA→B(∑ipiρi)=∑ipiNA→B(ρi)\mathcal{N}_{A\rightarrow B}(\sum_i p_i\rho_i)=\sum_ip_i\mathcal{N}_{A\rightarrow B}(\rho_i) NA→B​(i∑​pi​ρi​)=i∑​pi​NA→B​(ρi​)

  • Completely positive:

    NA→B(A)\mathcal{N}_{A\rightarrow B}(A)NA→B​(A) is positive for any positive (semi-) definite density operator AAA.

    Furthermore, if we introduce an extra system RRR of arbitrary dimensionality, it must be true that (I⊗NA→B)(A)(\mathcal{I}\otimes \mathcal{N}_{A\rightarrow B})(A)(I⊗NA→B​)(A) is positive (semi-) definite for any positive (semi-) definite operator AAA on the combined system RARARA, where I\mathcal{I}I is the identity map on HR\mathcal{H}_RHR​.

    Remark: for

    ∀A=(a11...a1k⋮⋱⋮ak1...akk)∈HRA,aij∈HA(I⊗NA→B)(A)=(NA→B(a11)...NA→B(a1k)⋮⋱⋮NA→B(ak1)...NA→B(akk))∈HRB\forall A = \begin{pmatrix} a_{11}&...&a_{1k}\\ \vdots&\ddots&\vdots\\ a_{k1}&...& a_{kk} \end{pmatrix}\in\mathcal{H}_{RA}, a_{ij}\in\mathcal{H}_A\\ (\mathcal{I}\otimes\mathcal{N}_{A\rightarrow B})(A)=\begin{pmatrix} \mathcal{N}_{A\rightarrow B}(a_{11})&...&\mathcal{N}_{A\rightarrow B}(a_{1k})\\ \vdots &\ddots &\vdots\\ \mathcal{N}_{A\rightarrow B}(a_{k1})&...&\mathcal{N}_{A\rightarrow B}(a_{kk}) \end{pmatrix}\in\mathcal{H}_{RB} ∀A=⎝⎜⎜⎛​a11​⋮ak1​​...⋱...​a1k​⋮akk​​⎠⎟⎟⎞​∈HRA​,aij​∈HA​(I⊗NA→B​)(A)=⎝⎜⎜⎛​NA→B​(a11​)⋮NA→B​(ak1​)​...⋱...​NA→B​(a1k​)⋮NA→B​(akk​)​⎠⎟⎟⎞​∈HRB​

  • Trace preserving:

    0≤tr(NA→B(ρ))≤10\leq tr(\mathcal{N}_{A\rightarrow B}(\rho))\leq 1 0≤tr(NA→B​(ρ))≤1

IF AND ONLY IF the map has a Kraus decomposition as follows:

NA→B(ρ)=∑l=0d−1VlρVl†\mathcal{N}_{A\rightarrow B}(\rho)=\sum_{l=0}^{d-1}V_l\rho V_l^\dagger NA→B​(ρ)=l=0∑d−1​Vl​ρVl†​

satisfying ∑l=0d−1Vl†Vl=I\sum_{l=0}^{d-1}V_l^\dagger V_l=I∑l=0d−1​Vl†​Vl​=I and d≤dim(HA)dim(HB)d\leq dim(\mathcal{H}_A)dim(\mathcal{H}_B)d≤dim(HA​)dim(HB​).

Proof:

  • If the map has the form:

    NA→B(ρ)=∑l=0d−1VlρVl†\mathcal{N}_{A\rightarrow B}(\rho)=\sum_{l=0}^{d-1}V_l\rho V_l^\dagger NA→B​(ρ)=l=0∑d−1​Vl​ρVl†​

    • Convex linear:

      NA→B(∑ipiρi)=∑l=0d−1Vl∑ipiρiVl†=∑ipi∑l=0d−1VlρiVl†=∑ipiNA→B(ρi)\begin{aligned} \mathcal{N}_{A\rightarrow B}(\sum_ip_i\rho_i)&=\sum_{l=0}^{d-1}V_l\sum_{i}p_i\rho_i V_l^\dagger\\ &=\sum_ip_i\sum_{l=0}^{d-1}V_l\rho_i V_l^\dagger\\ &=\sum_ip_i\mathcal{N}_{A\rightarrow B}(\rho_i) \end{aligned} NA→B​(i∑​pi​ρi​)​=l=0∑d−1​Vl​i∑​pi​ρi​Vl†​=i∑​pi​l=0∑d−1​Vl​ρi​Vl†​=i∑​pi​NA→B​(ρi​)​

    • Complete positive:

      For any valid density operator ρ∈HA\rho\in\mathcal{H}_Aρ∈HA​, it’s positive (semi-) definite. Then:

      ∀∣ψ⟩∈HRB,⟨ψ∣(I⊗NA→B)(ρ)∣ψ⟩=⟨ψ∣∑l=0d−1(I⊗Vl)ρ(I⊗Vl†)∣ψ⟩=⟨ψ∣∑l=0d−1(I⊗Vl)ρ(I⊗Vl)†∣ψ⟩=∑l=0d−1[⟨ψ∣(I⊗Vl)]⋅ρ⋅[(I⊗Vl)†∣ψ⟩]=∑l=0d−1≥0≥0\begin{aligned} \forall |\psi\rangle\in\mathcal{H}_{RB},\\ \langle\psi|(\mathcal{I}\otimes \mathcal{N}_{A\rightarrow B})(\rho)|\psi\rangle &=\langle\psi|\sum_{l=0}^{d-1}(I\otimes V_l)\rho(I\otimes V_l^\dagger)|\psi\rangle\\ &=\langle\psi|\sum_{l=0}^{d-1}(I\otimes V_l)\rho(I\otimes V_l)^\dagger|\psi\rangle\\ &=\sum_{l=0}^{d-1}[\langle\psi|(I\otimes V_l)]\cdot\rho\cdot[(I\otimes V_l)^\dagger|\psi\rangle]\\ &=\sum_{l=0}^{d-1}\geq 0\\ &\geq 0 \end{aligned} ∀∣ψ⟩∈HRB​,⟨ψ∣(I⊗NA→B​)(ρ)∣ψ⟩​=⟨ψ∣l=0∑d−1​(I⊗Vl​)ρ(I⊗Vl†​)∣ψ⟩=⟨ψ∣l=0∑d−1​(I⊗Vl​)ρ(I⊗Vl​)†∣ψ⟩=l=0∑d−1​[⟨ψ∣(I⊗Vl​)]⋅ρ⋅[(I⊗Vl​)†∣ψ⟩]=l=0∑d−1​≥0≥0​

      It’s because ρ\rhoρ is positive (semi-) definite, so for ∣φ⟩=(I⊗Vl)†∣ψ⟩|\varphi\rangle=(I\otimes V_l)^\dagger|\psi\rangle∣φ⟩=(I⊗Vl​)†∣ψ⟩, ⟨φ∣ρ∣φ⟩≥0\langle\varphi|\rho|\varphi\rangle\geq 0⟨φ∣ρ∣φ⟩≥0.

    • Trace preserving:

      tr(NA→B(ρ))=tr(∑l=0d−1VlρVl†)=tr(ρ∑l=0d−1Vl†Vl)=tr(ρ)\begin{aligned} tr(\mathcal{N}_{A\rightarrow B}(\rho))&=tr(\sum_{l=0}^{d-1}V_l\rho V_l^\dagger)\\ &=tr(\rho\sum_{l=0}^{d-1}V_l^\dagger V_l)\\ &=tr(\rho) \end{aligned} tr(NA→B​(ρ))​=tr(l=0∑d−1​Vl​ρVl†​)=tr(ρl=0∑d−1​Vl†​Vl​)=tr(ρ)​

      where we use tr(AB)=tr(BA)tr(AB)=tr(BA)tr(AB)=tr(BA).

  • On the other hand, if the map is linear, completely positive, trace preserving, we prove that the map has Kraus decomposition.

    Let dA≡dim(HA),dB≡dim(HB)d_A\equiv dim(\mathcal{H}_A),d_B\equiv dim(\mathcal{H}_B)dA​≡dim(HA​),dB​≡dim(HB​) and RRR is an auxiliary system with the same orthonormal basis with AAA. Let ∣Γ⟩RA|\Gamma\rangle_{RA}∣Γ⟩RA​ denote the following unnormalized maximally entangled vector:

    ∣Γ⟩RA≡∑i=0dA−1∣i⟩R⊗∣i⟩A|\Gamma\rangle_{RA}\equiv\sum_{i=0}^{d_A-1}|i\rangle_R\otimes |i\rangle_A ∣Γ⟩RA​≡i=0∑dA​−1​∣i⟩R​⊗∣i⟩A​

    then we have

    (I⊗NA→B)(∣Γ⟩⟨Γ∣RA)=∑i,j=0dA−1∣i⟩⟨j∣R⊗NA→B(∣i⟩⟨j∣A)(\mathcal{I}\otimes\mathcal{N}_{A\rightarrow B})(|\Gamma\rangle\langle\Gamma|_{RA})=\sum_{i,j=0}^{d_A-1}|i\rangle\langle j|_R\otimes\mathcal{N}_{A\rightarrow B}(|i\rangle\langle j|_A) (I⊗NA→B​)(∣Γ⟩⟨Γ∣RA​)=i,j=0∑dA​−1​∣i⟩⟨j∣R​⊗NA→B​(∣i⟩⟨j∣A​)

    Since ∣Γ⟩⟨Γ∣RA|\Gamma\rangle\langle \Gamma|_{RA}∣Γ⟩⟨Γ∣RA​ is positive semi-definite, since NA→B\mathcal{N}_{A\rightarrow B}NA→B​ is completely positive, the RHS is positive semi-definite, too. So it can be diagonalized:

    ∑i,j=0dA−1∣i⟩⟨j∣R⊗NA→B(∣i⟩⟨j∣A)=∑l=0d−1∣ϕl⟩⟨ϕl∣RB\sum_{i,j=0}^{d_A-1}|i\rangle\langle j|_R\otimes\mathcal{N}_{A\rightarrow B}(|i\rangle\langle j|_A)=\sum_{l=0}^{d-1}|\phi_l\rangle\langle\phi_l|_{RB} i,j=0∑dA​−1​∣i⟩⟨j∣R​⊗NA→B​(∣i⟩⟨j∣A​)=l=0∑d−1​∣ϕl​⟩⟨ϕl​∣RB​

    where d≤dAdBd\leq d_Ad_Bd≤dA​dB​, dAdBd_Ad_BdA​dB​ is the dimension of system RBRBRB, and ∣ϕl⟩|\phi_l\rangle∣ϕl​⟩ does not necessarily have to be orthonormal.

    we expand the ∣ϕl⟩|\phi_l\rangle∣ϕl​⟩ in terms of an orthonormal basis of RBRBRB:

    ∣ϕ⟩RB=∑i=0dA−1∑j=0dB−1αij∣i⟩R⊗∣j⟩B|\phi\rangle_{RB}=\sum_{i=0}^{d_A-1}\sum_{j=0}^{d_B-1}\alpha_{ij}|i\rangle_R\otimes|j\rangle_B ∣ϕ⟩RB​=i=0∑dA​−1​j=0∑dB​−1​αij​∣i⟩R​⊗∣j⟩B​

    let VA→B\mathcal{V}_{A\rightarrow B}VA→B​ denote the following linear operator:

    VA→B≡∑i=0dA−1∑j=0dB−1αi,j∣j⟩B⟨i∣A\mathcal{V}_{A\rightarrow B}\equiv\sum_{i=0}^{d_A-1}\sum_{j=0}^{d_B-1}\alpha_{i,j}|j\rangle_B\langle i|_A VA→B​≡i=0∑dA​−1​j=0∑dB​−1​αi,j​∣j⟩B​⟨i∣A​

    Then we see:

    (I⊗VA→B)∣Γ⟩RA=(I⊗∑i=0dA−1∑j=0dB−1αi,j∣j⟩B⟨i∣A)(∑k=0dA−1∣k⟩R⊗∣k⟩A)=∑i=0dA−1∑j=0dB−1∑k=0dA−1αi,j∣k⟩R⊗∣j⟩B⟨i∣k⟩A=∑i=0dA−1∑j=0dB−1αi,j∣i⟩R⊗∣j⟩B=∣ϕ⟩RB\begin{aligned} (\mathcal{I}\otimes\mathcal{V}_{A\rightarrow B})|\Gamma\rangle_{RA}&=(I\otimes\sum_{i=0}^{d_A-1}\sum_{j=0}^{d_B-1}\alpha_{i,j}|j\rangle_B\langle i|_A)(\sum_{k=0}^{d_A-1}|k\rangle_R\otimes|k\rangle_A)\\ &=\sum_{i=0}^{d_A-1}\sum_{j=0}^{d_B-1}\sum_{k=0}^{d_A-1}\alpha_{i,j}|k\rangle_R\otimes|j\rangle_B\langle i|k\rangle_A\\ &=\sum_{i=0}^{d_A-1}\sum_{j=0}^{d_B-1}\alpha_{i,j}|i\rangle_R\otimes|j\rangle_B\\ &=|\phi\rangle_{RB} \end{aligned} (I⊗VA→B​)∣Γ⟩RA​​=(I⊗i=0∑dA​−1​j=0∑dB​−1​αi,j​∣j⟩B​⟨i∣A​)(k=0∑dA​−1​∣k⟩R​⊗∣k⟩A​)=i=0∑dA​−1​j=0∑dB​−1​k=0∑dA​−1​αi,j​∣k⟩R​⊗∣j⟩B​⟨i∣k⟩A​=i=0∑dA​−1​j=0∑dB​−1​αi,j​∣i⟩R​⊗∣j⟩B​=∣ϕ⟩RB​​

    So we see that for all vector ∣ϕ⟩RB|\phi\rangle_{RB}∣ϕ⟩RB​, we can find a linear operator VA→B\mathcal{V}_{A\rightarrow B}VA→B​ such that (I⊗VA→B)∣Γ⟩RA=∣ϕ⟩RB(\mathcal{I}\otimes\mathcal{V}_{A\rightarrow B})|\Gamma\rangle_{RA}=|\phi\rangle_{RB}(I⊗VA→B​)∣Γ⟩RA​=∣ϕ⟩RB​.

    And we have:

    (⟨i∣R⊗I)∣ϕ⟩RB=(⟨i∣R⊗I)⋅(I⊗VA→B)∣Γ⟩RA=(⟨i∣R⊗I)(I⊗VA→B)∑j=0dA−1∣j⟩R⊗∣j⟩A=∑j=0dA−1⟨i∣j⟩R⊗VA→B∣j⟩A=VA→B∣i⟩A\begin{aligned} (\langle i|_R\otimes I)|\phi\rangle_{RB}&=(\langle i|_R\otimes I)\cdot(\mathcal{I}\otimes\mathcal{V}_{A\rightarrow B})|\Gamma\rangle_{RA}\\ &=(\langle i|_R\otimes I)(\mathcal{I}\otimes\mathcal{V}_{A\rightarrow B})\sum_{j=0}^{d_A-1}|j\rangle_R\otimes|j\rangle_A\\ &=\sum_{j=0}^{d_A-1}\langle i|j\rangle_R\otimes \mathcal{V}_{A\rightarrow B}|j\rangle_A\\ &=\mathcal{V}_{A\rightarrow B}|i\rangle_A \end{aligned} (⟨i∣R​⊗I)∣ϕ⟩RB​​=(⟨i∣R​⊗I)⋅(I⊗VA→B​)∣Γ⟩RA​=(⟨i∣R​⊗I)(I⊗VA→B​)j=0∑dA​−1​∣j⟩R​⊗∣j⟩A​=j=0∑dA​−1​⟨i∣j⟩R​⊗VA→B​∣j⟩A​=VA→B​∣i⟩A​​

    Finally we have:

    NA→B(∣i⟩⟨j∣A)=(⟨i∣R⊗I)⋅∑p,q=0dA−1∣p⟩⟨q∣R⊗NA→B(∣p⟩⟨q∣A)⋅(∣j⟩R⊗I)=(⟨i∣R⊗I)⋅∑l=0d−1∣ϕl⟩⟨ϕl∣RB⋅(∣j⟩R⊗I)=∑l=0d−1(⟨i∣R⊗I)∣ϕl⟩⋅⟨ϕl∣(∣j⟩R⊗I)=∑l=0d−1(Vl)A→B∣i⟩⟨j∣A(Vl)A→B†\begin{aligned} \mathcal{N}_{A\rightarrow B}(|i\rangle\langle j|_A)&=(\langle i|_R\otimes I)\cdot \sum_{p,q=0}^{d_A-1}|p\rangle\langle q|_R\otimes\mathcal{N}_{A\rightarrow B}(|p\rangle\langle q|_A)\cdot(|j\rangle_R\otimes I)\\ &=(\langle i|_R\otimes I)\cdot\sum_{l=0}^{d-1}|\phi_l\rangle\langle\phi_l|_{RB}\cdot(|j\rangle_R\otimes I)\\ &=\sum_{l=0}^{d-1}(\langle i|_R\otimes I)|\phi_l\rangle\cdot\langle\phi_l|(|j\rangle_R\otimes I)\\ &=\sum_{l=0}^{d-1}(\mathcal{V}_l)_{A\rightarrow B}|i\rangle\langle j|_A(\mathcal{V}_l)_{A\rightarrow B}^\dagger \end{aligned} NA→B​(∣i⟩⟨j∣A​)​=(⟨i∣R​⊗I)⋅p,q=0∑dA​−1​∣p⟩⟨q∣R​⊗NA→B​(∣p⟩⟨q∣A​)⋅(∣j⟩R​⊗I)=(⟨i∣R​⊗I)⋅l=0∑d−1​∣ϕl​⟩⟨ϕl​∣RB​⋅(∣j⟩R​⊗I)=l=0∑d−1​(⟨i∣R​⊗I)∣ϕl​⟩⋅⟨ϕl​∣(∣j⟩R​⊗I)=l=0∑d−1​(Vl​)A→B​∣i⟩⟨j∣A​(Vl​)A→B†​​

    We still need to prove that ∑lVl†Vl=I\sum_{l}\mathcal{V}_l^\dagger\mathcal{V}_l=I∑l​Vl†​Vl​=I. Since NA→B\mathcal{N}_{A\rightarrow B}NA→B​ is trace preserving, then:

    tr(NA→B(∣i⟩⟨j∣A))=tr(∣i⟩⟨j∣A)=δijtr(\mathcal{N}_{A\rightarrow B}(|i\rangle\langle j|_A))=tr(|i\rangle\langle j|_A)=\delta_{ij} tr(NA→B​(∣i⟩⟨j∣A​))=tr(∣i⟩⟨j∣A​)=δij​

    Consider:

    δij=tr(NA→B(∣i⟩⟨j∣A))=tr(∑lVl∣i⟩⟨j∣AVl†)=tr(∑lVl†Vl∣i⟩⟨j∣A)=⟨j∣A∑lVl†Vl∣i⟩A\begin{aligned} \delta_{ij}&=tr(\mathcal{N}_{A\rightarrow B}(|i\rangle\langle j|_A))\\ &=tr(\sum_{l}\mathcal{V}_l|i\rangle\langle j|_A\mathcal{V}_l^\dagger)\\ &=tr(\sum_l\mathcal{V}_l^\dagger\mathcal{V}_l|i\rangle\langle j|_A)\\ &=\langle j|_A\sum_l\mathcal{V}_l^\dagger\mathcal{V}_l|i\rangle_A\\ \end{aligned} δij​​=tr(NA→B​(∣i⟩⟨j∣A​))=tr(l∑​Vl​∣i⟩⟨j∣A​Vl†​)=tr(l∑​Vl†​Vl​∣i⟩⟨j∣A​)=⟨j∣A​l∑​Vl†​Vl​∣i⟩A​​

    That is, ∑lVl†Vl=I\sum_l\mathcal{V}_l^\dagger\mathcal{V}_l=I∑l​Vl†​Vl​=I.
