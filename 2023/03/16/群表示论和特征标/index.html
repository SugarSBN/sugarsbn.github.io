因为两个都是子模。这也揭示了给定一个子模UUU，我们想找另一个子模WWW，那么就可以先任取一个子空间WWW 使得V=U⊕WV=U\oplus WV=U⊕W，那么对于空间U⊕WU\oplus WU⊕W 的一组基B\mathscr{B}B 就会有形式

[g]B=(∗0∗∗)[g]_\mathscr{B}= \left ( \begin{array}{c|c} *& 0\\ \hline *&* \end{array} \right ) [g]B​=(∗∗​0∗​​)

然后我们作矩阵变换使得对于所有的ggg 让左下角也是 0 即可。（实际上只要对生成元的ggg，在循环群情况下）

定理：每一个非零 FG - 模都是若干个不可约 FG - 模的直和。

V=U1⊕...⊕UnV=U_1\oplus ...\oplus U_n V=U1​⊕...⊕Un​

就是反复应用 Maschke 定理的结果。

# Schur 引理

Schur 引理：假设V,WV,WV,W 是两个不可约CG\mathbb{C}GCG - 模。那么：

  • 若ϑ:V→W\vartheta:V\rightarrow Wϑ:V→W 是一个CG\mathbb{C}GCG 同态，那么它要么是一个同构，要么有∀v∈V,ϑ(v)=0\forall v\in V,\vartheta(v)=0∀v∈V,ϑ(v)=0。
  • 若ϑ:V→W\vartheta:V\rightarrow Wϑ:V→W 是一个CG\mathbb{C}GCG 同构，则ϑ\varthetaϑ 是一个数乘变换。

定理：若VVV 是一个非零CG\mathbb{C}GCG - 模，并且每一个V→VV\rightarrow VV→V 的CG\mathbb{C}GCG 同态都是一个数乘变换，那么VVV 不可约。

推论：令ρ:G→GL(n,C)\rho:G\rightarrow GL(n,\mathbb{C})ρ:G→GL(n,C) 是群GGG 的一个表示，那么ρ\rhoρ 不可约当且仅当每一个满足：

∀g∈G,ρ(g)A=Aρ(g)\forall g\in G,\rho(g)A=A\rho(g) ∀g∈G,ρ(g)A=Aρ(g)

的矩阵AAA 都是数乘矩阵，即A=λIA=\lambda IA=λI。

例：令G={1,a,a2},a3=1G=\{1,a,a^2\},a^3=1G={1,a,a2},a3=1，且ρ:G→GL(2,C)\rho:G\rightarrow GL(2,\mathbb{C})ρ:G→GL(2,C) 是一个表示，为ρ(a)=(01−1−1)\rho(a)=\begin{pmatrix}0&1\\-1&-1\end{pmatrix}ρ(a)=(0−1​1−1​)。由于群是交换的，那么令A=ρ(a)A=\rho(a)A=ρ(a)，那么∀g,ρ(g)A=Aρ(g)\forall g,\rho(g)A=A\rho(g)∀g,ρ(g)A=Aρ(g)，且A=ρ(a)A=\rho(a)A=ρ(a) 不是数乘矩阵，故该表示是可约的。

这里 comment 一下我对表示可约的理解。群表示实际上是把群结构同态映射到了一个线性空间上，即一个nnn 维矩阵。这个矩阵有什么含义呢？

我们把它看作一个基变换矩阵来看，譬如对于一个nnn 维向量空间VVV，我们可以找到它的一组nnn 维向量形式的基（同构意义下），即：

∀v∈V,v=λ1v1+...+λnvn,vi∈Cn\forall v\in V,v=\lambda_1v_1+...+\lambda_nv_n,v_i\in\mathbb{C}^n ∀v∈V,v=λ1​v1​+...+λn​vn​,vi​∈Cn

那么就有ρ(g):V→V\rho(g):V\rightarrow Vρ(g):V→V，即自然定义了一个CG\mathbb{C}GCG - 模：

ρ(g)(v)=ρ(g)⋅∑i=1nλivi=∑i=1nλiρ(g)vi∈V,ρ(g)∈Cn×n,v∈V\rho(g)(v)=\rho(g)\cdot \sum_{i=1}^n\lambda_iv_i=\sum_{i=1}^n\lambda_i\rho(g)v_i\in V,\\ \rho(g)\in\mathbb{C}^{n\times n},v\in V ρ(g)(v)=ρ(g)⋅i=1∑n​λi​vi​=i=1∑n​λi​ρ(g)vi​∈V,ρ(g)∈Cn×n,v∈V

那么什么时候表示ρ\rhoρ 是可约的呢？那就是存在两个子空间V1,V2V_1,V_2V1​,V2​ 使得V1⊗V2=VV_1\otimes V_2=VV1​⊗V2​=V，然后对于每个ggg，都可以把ρ(g)\rho(g)ρ(g) 等价变换为如下形式：

ρ(g)=(∗00∗)\rho(g)= \left( \begin{array}{c|c} *&0\\ \hline 0&* \end{array} \right) ρ(g)=(∗0​0∗​​)

其中左上角为dimV1×dimV1dim V_1\times dimV_1dimV1​×dimV1​ 的矩阵，右下角是dimV2×dimV2dim V_2\times dimV_2dimV2​×dimV2​ 的矩阵。此时我们可以把表示实际上拆成两个子空间上分别的基变换矩阵的表示，这和CG\mathbb{C}GCG - 模可约定的定义是一致的。

定理：对于某个有限交换群，那么它的每一个不可与CG\mathbb{C}GCG - 模均是一维的。

# 特征标

定义：假设VVV 是一个基为B\mathscr{B}B 的CG\mathbb{C}GCG - 模，那么VVV 的特征标定义为如下函数：

χ(g)=tr[g]B\chi(g)=tr[g]_{\mathscr{B}} χ(g)=tr[g]B​

很显然，对同一个CG\mathbb{C}GCG - 模，即使选择的基不一样，那也有[g]B=T−1[g]B′T[g]_{\mathscr{B}}=T^{-1}[g]_{\mathscr{B}'}T[g]B​=T−1[g]B′​T，故取 trace 后都是相等的。

同样，我们可以自然地定义表示ρ\rhoρ 对应的特征标。因为ρ(g)\rho(g)ρ(g) 是可逆矩阵，那么它也可以看作是一个基变换矩阵（一致于先前对可约性的说明），那么就可以自然地定义χ(g)=tr(ρ(g))\chi(g)=tr(\rho(g))χ(g)=tr(ρ(g))。

那么自然地，我们也可以定义特征标的可约性：

定义：一个特征标是可约的，当且仅当它是一个可约的CG\mathbb{C}GCG - 模的特征标。

定理：同构的CG\mathbb{C}GCG - 模有相同的特征标。共轭元有相同的特征标。即

tr[x]B=tr[g−1yg]B=tr([g]B−1[y]B[g]B)=tr[y]Btr[x]_{\mathscr{B}}=tr[g^{-1}yg]_{\mathscr{B}}=tr([g]_{\mathscr{B}}^{-1}[y]_{\mathscr{B}}[g]_{\mathscr{B}})=tr[y]_{\mathscr{B}} tr[x]B​=tr[g−1yg]B​=tr([g]B−1​[y]B​[g]B​)=tr[y]B​

接下来的结论揭示了特征标的重要性，说明了我们要确定一个表示的核只需要 知道它的特征标信息就可以了：

命题：令ρ:G→GL(n,C)\rho:G\rightarrow GL(n,\mathbb{C})ρ:G→GL(n,C) 为GGG 的一个表示，χ\chiχ 是ρ\rhoρ 的一个特征标，那么有：

Kerρ={g∈G  ∣  χ(g)=χ(1)=n}Ker\rho = \{g\in G\;|\;\chi(g)=\chi(1)=n\} Kerρ={g∈G∣χ(g)=χ(1)=n}

所以我们往往定义特征标的核Kerχ={g∈G  ∣  χ(g)=χ(1)}Ker\chi=\{g\in G\;|\;\chi(g)=\chi(1)\}Kerχ={g∈G∣χ(g)=χ(1)}。自然地，一个正则的特征标也就定义为正则CG\mathbb{C}GCG - 模上的特征标了。

定理：若χreg\chi_{reg}χreg​ 是群GGG 的正则特征标，那么：

{χreg(1)=∣G∣χreg(g)=0,g≠1\begin{cases} \chi_{reg}(1)=|G|\\ \chi_{reg}(g)=0,g\neq 1 \end{cases} {χreg​(1)=∣G∣χreg​(g)=0,g=1​

这个定理很好证明。因为群的正则CG\mathbb{C}GCG - 模的一组基就是群GGG 中的所有元素，故对于某个基向量gi∈Gg_i\in Ggi​∈G，那么存在gj∈Gg_j\in Ggj​∈G 使得gig=gjg_ig=g_jgi​g=gj​。故在[g][g][g] 中，第iii 行有且只有第jjj 列为 1，其他都为 0。

故形象地理解下就知道，只有g=1g=1g=1 的话，[g]=I[g]=I[g]=I，否则[g][g][g] 的对角线元素均为 0。

定义：设ϑ,ϕ\vartheta,\phiϑ,ϕ 是G→CG\rightarrow\mathbb{C}G→C 的函数，那么如下定义它们的内积：

⟨ϑ,ϕ⟩=1∣G∣∑g∈Gϑ(g)ϕ(g)∗\langle\vartheta,\phi\rangle=\frac{1}{|G|}\sum_{g\in G}\vartheta(g)\phi(g)^* ⟨ϑ,ϕ⟩=∣G∣1​g∈G∑​ϑ(g)ϕ(g)∗

所以自然把特征标的内积定义为⟨χ1,χ2⟩=1∣G∣∑g∈Gχ1(g)χ2(g−1)\langle\chi_1,\chi_2\rangle=\frac{1}{|G|}\sum_{g\in G}\chi_1(g)\chi_2(g^{-1})⟨χ1​,χ2​⟩=∣G∣1​∑g∈G​χ1​(g)χ2​(g−1)。

特征标主定理：令U,VU,VU,V 是非同构、不可约的CG\mathbb{C}GCG - 模，它们的特征标分别是χ,ψ\chi,\psiχ,ψ，那么：

⟨χ,χ⟩=1,⟨χ,ψ⟩=0\langle\chi,\chi\rangle=1,\langle\chi,\psi\rangle=0 ⟨χ,χ⟩=1,⟨χ,ψ⟩=0
