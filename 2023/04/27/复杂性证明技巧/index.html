# HALT 的不可判定性

Theorem: 下述语言是不可判定的：

LHALT={⟨α,x⟩  ∣  Mα  halts  on  input  x}L_{HALT}=\{\langle \alpha,x\rangle\;|\;M_\alpha\;halts\;on\;input\;x\} LHALT​={⟨α,x⟩∣Mα​haltsoninputx}

证明：使用对角线方法（Diagonalization），即考虑M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) 的情况，着手证明。

首先，考虑下列函数：

UC(x)={0if  Mx(x)  halts  and  output  11otherwiseUC(x)=\begin{cases} 0 & if\;M_x(x)\;halts\;and\;output\;1\\ 1 &otherwise \end{cases} UC(x)={01​ifMx​(x)haltsandoutput1otherwise​

那么先证明下述语言是不可判定的：

LUC={x  ∣  UC(x)=1}L_{UC}=\{x\;|\;UC(x)=1\} LUC​={x∣UC(x)=1}

反设存在一个图灵机MMM 判定LUCL_{UC}LUC​，那么我们考虑MMM 在输入⟨M⟩\langle M\rangle⟨M⟩ 上会怎样：

  • 若M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) 停机了。

    • 若M(⟨M⟩)=0M(\langle M\rangle)=0M(⟨M⟩)=0，由于MMM 判定LUCL_{UC}LUC​，故⟨M⟩∉LUC\langle M\rangle\notin L_{UC}⟨M⟩∈/LUC​，即UC(⟨M⟩)=0UC(\langle M\rangle)=0UC(⟨M⟩)=0。

      根据UCUCUC 函数的定义，UC(⟨M⟩)=0UC(\langle M\rangle)=0UC(⟨M⟩)=0 当且仅当M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) “halts and output 1”，矛盾。

    • 若M(⟨M⟩)=1M(\langle M\rangle)=1M(⟨M⟩)=1，由于MMM 判定LUCL_{UC}LUC​，故⟨M⟩∈LUC\langle M\rangle\in L_{UC}⟨M⟩∈LUC​，即UC(⟨M⟩)=1UC(\langle M\rangle)=1UC(⟨M⟩)=1。

      根据UCUCUC 函数的定义，UC(⟨M⟩)=1UC(\langle M\rangle)=1UC(⟨M⟩)=1 当且仅当M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) 不停机或者输出 0。由于假设M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) 停机了，故M(⟨M⟩)=0M(\langle M\rangle)=0M(⟨M⟩)=0，矛盾。

  • 若M(⟨M⟩)M(\langle M\rangle)M(⟨M⟩) 不停机，与 “MMM 判断LUCL_{UC}LUC​” 矛盾，即有输入⟨M⟩\langle M\rangle⟨M⟩ 导致MMM 无法判断。

证毕。

这个定理我的理解其实是 “任何尝试去判定LUCL_{UC}LUC​ 的图灵机MMM，都一定会存在某个输入input=⟨M⟩input=\langle M\rangleinput=⟨M⟩，使得MMM 不停机，故无法判断它。”

下面我们证明LHALTL_{HALT}LHALT​ 的不可判定性。其实证明本质是，一旦我们可以提前得知是否停机，我们就可以判定LUCL_{UC}LUC​，而LUCL_{UC}LUC​ 的不可判定性也恰恰来自于是否停机。

反设存在一个图灵机MMM 判定LHALTL_{HALT}LHALT​，那么我们可以构造一个图灵机M′M'M′ 去判定LUCL_{UC}LUC​：

  • 对于输入xxx，使用M(⟨x,x⟩)M(\langle x,x\rangle)M(⟨x,x⟩) 去判定Mx(x)M_x(x)Mx​(x) 是否停机。
    • 若不停机，则M′M'M′ 直接输出 1。
    • 若停机，则让M′M'M′ 去模拟MxM_xMx​，并输出与Mx(x)M_x(x)Mx​(x) 相反的结果。

很显然，上述图灵机M′M'M′ 判定了LUCL_{UC}LUC​，矛盾。

故证毕。

# Cook-Levin Theorem: Boolean Circuit 的通用性

Theorem (Cook-Levin):

CIRCUIT−SAT∈NP−CompleteCIRCUIT-SAT\in NP-Complete CIRCUIT−SAT∈NP−Complete

Proof：首先CIRCUIT−SAT∈NPCIRCUIT-SAT\in NPCIRCUIT−SAT∈NP 是显而易见的，所以我们证明的重点在于，∀L∈NP,L≤mpCIRCUIT−SAT\forall L\in NP,L\leq_m^p CIRCUIT-SAT∀L∈NP,L≤mp​CIRCUIT−SAT。

这个定理初看非常困难，因为要证明∀L∈NP\forall L\in NP∀L∈NP。但实际上，根据NPNPNP 问题的定义，实际上就是要证明图灵机向 Boolean Circuit 的规约。

考虑L∈NPL\in NPL∈NP，故存在一个判定图灵机VVV，使得∀x∈L,∃y,V(x,y)=1\forall x\in L,\exists y,V(x,y)=1∀x∈L,∃y,V(x,y)=1。

# P=NP⇒EXP=NEXPP=NP\Rightarrow EXP=NEXPP=NP⇒EXP=NEXP 的证明技巧: Padding

Theorem:

P=NP⇒EXP=NEXPP=NP\Rightarrow EXP=NEXP P=NP⇒EXP=NEXP

Proof：假设P=NPP=NPP=NP。EXP⊆NEXPEXP\subseteq NEXPEXP⊆NEXP 是显然的，我们着手证明NEXP⊆EXPNEXP\subseteq EXPNEXP⊆EXP。

对于L∈NEXPL\in NEXPL∈NEXP，我们人为地构造一个语言：

Lpad={⟨x,12∣x∣k⟩  ∣  x∈L}L_{pad}=\{\langle x,1^{2^{|x|^k}}\rangle\;|\;x\in L\} Lpad​={⟨x,12∣x∣k⟩∣x∈L}

其实就是对LLL 中的字符串，后面加一个长度为2∣x∣k2^{|x|^k}2∣x∣k，全为 1 的 padding。kkk 对证明不是很重要。

我们可以证明，padding 后的这个语言Lpad∈NPL_{pad}\in NPLpad​∈NP。假设存在一个非确定性图灵机MMM 去判定LLL。

我们构造一个图灵机M′M'M′ 去判定LpadL_{pad}Lpad​：

  1. 对于输入yyy，判断yyy 的形式是否满足y=⟨x,12∣x∣k⟩y=\langle x,1^{2^{|x|^k}}\rangley=⟨x,12∣x∣k⟩，若不是，则拒绝。

    记n=∣y∣=O(2∣x∣k)n=|y|=O(2^{|x|^k})n=∣y∣=O(2∣x∣k)，那么这一步需要的复杂度为O(poly(n))O(poly(n))O(poly(n))。

  2. 提取出xxx，然后去模拟非确定性图灵机MMM。并输出MMM 的结果。由于L∈NEXPL\in NEXPL∈NEXP，故MMM 所需步骤为O(poly(∣x∣)2∣x∣k)=O(poly(logn)⋅n)O(poly(|x|)2^{|x|^k})=O(poly(logn)\cdot n)O(poly(∣x∣)2∣x∣k)=O(poly(logn)⋅n)，对于M′M'M′ 来说，就是多项式级别。

故综上，Lpad∈NPL_{pad}\in NPLpad​∈NP。因为P=NPP=NPP=NP，故Lpad∈PL_{pad}\in PLpad​∈P，故存在一个确定型图灵机M∗M^*M∗ 判定LpadL_{pad}Lpad​。下面我们证明L∈EXPL\in EXPL∈EXP。

构造这样一个图灵机：

  1. 对于输入xxx，先 pad 2∣x∣k2^{|x|^k}2∣x∣k 个 1。这一步复杂度为O(2∣x∣k)O(2^{|x|^k})O(2∣x∣k)。
  2. 模拟M∗M^*M∗ 去判定xxx。这一步复杂度也为指数级。

综上，L∈EXPL\in EXPL∈EXP。证毕
