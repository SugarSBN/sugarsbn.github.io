# 曲线、曲面与多项式

对空间中的一条曲线或曲面，我们往往用参数方程来表示：

p(u)=(x(u)y(u)z(u))p(u,v)=(x(u,v)y(u,v)z(u,v))\mathbf{p}(u)=\begin{pmatrix} x(u)\\ y(u)\\ z(u) \end{pmatrix} \quad \mathbf{p}(u,v)=\begin{pmatrix} x(u,v)\\ y(u,v)\\ z(u,v) \end{pmatrix} p(u)=⎝⎛​x(u)y(u)z(u)​⎠⎞​p(u,v)=⎝⎛​x(u,v)y(u,v)z(u,v)​⎠⎞​

我们往往认为u,vu,vu,v 是一个从 0 变化到 1 的实数标量，那么前者p(u)\mathbf{p}(u)p(u) 是一条曲线，后者p(u,v)\mathbf{p}(u,v)p(u,v) 是一个曲面。

然而这是一般情况。在实际应用中，相比于一般函数x(u),y(u),z(u)x(u),y(u),z(u)x(u),y(u),z(u)，我们更关心它们是多项式时的情况。参数方程改写为：

p(u)=∑k=0nukckp(u,v)=∑k=0n∑l=0mukvlck,l\mathbf{p}(u)=\sum_{k=0}^n u^k\mathbf{c}_k\quad \mathbf{p}(u,v)=\sum_{k=0}^n\sum_{l=0}^m u^k v^l\mathbf{c}_{k,l} p(u)=k=0∑n​ukck​p(u,v)=k=0∑n​l=0∑m​ukvlck,l​

其中ck,ck,l\mathbf{c}_{k},\mathbf{c}_{k,l}ck​,ck,l​ 都是三维的点或向量，严格来说c0\mathbf{c}_0c0​ 应为点，其余是向量。

# 插值

插值问题就是给出一组点p0,p1,...,pn\mathbf{p}_0,\mathbf{p}_1,...,\mathbf{p}_np0​,p1​,...,pn​，要求你构造一条曲线去经过这些点。实际中往往采用三次函数来插值，因为二次往往有些欠拟合，而更高次又增加了计算量，是一个经验的折中。

{0,0,0,0,1,2,...,n−1,n,n,n,n}\{0,0,0,0,1,2,...,n-1,n,n,n,n\} {0,0,0,0,1,2,...,n−1,n,n,n,n}

n−(m+1)=dn-(m+1)=d n−(m+1)=d

  1. 给出控制点p0,...,pm\mathbf{p}_0,...,\mathbf{p}_mp0​,...,pm​，以及多项式次数ddd。
  2. 选取结点{u0,...,ud+m+1}\{u_0,...,u_{d+m+1}\}{u0​,...,ud+m+1​}。
  3. 计算出 basis 多项式{Bid(u)}\{B_{id}(u)\}{Bid​(u)}。
  4. 计算出曲线p(u)=∑i=0mBid(u)pi\mathbf{p}(u)=\sum_{i=0}^m B_{id}(u)\mathbf{p}_ip(u)=∑i=0m​Bid​(u)pi​。
