# 关于目前

刚刚考完 CSP 心情低落删除了之前博客上所有关于 OI 的文章，然后整理了下这个网站，辣鸡 DNSpod 解析巨慢等我有钱必换，发现换了 hexo 手机无法访问又不想更 IOS13…

然后高三期中考试刚过，马上就是一阵接一阵的考试，让人完全无法停下来想一下自己想想的东西… 真的是人菜事情多。

然后这个博客估计以后也是随缘更新的状态，因为没了技术内容日常也没啥好更新的，或者更新点摘抄？尝试下把 kindle 剪贴导出来看看，然后主要是自己使用，可惜就是手机看不了，越想越气。
