# The First Sylow Theorem

The first Sylow theorem can be viewed as a specific converse of the Lagrange theorem.

(Sylow I) Given a group GGG such that

∣G∣=n=pem,|G|=n=p^em, ∣G∣=n=pem,

where pep^epe is the largest power of ppp (that is, gcd(p,m)=1gcd(p,m)=1gcd(p,m)=1), then there exists a subgroup H≤GH\leq GH≤G such that

∣H∣=pe.|H|=p^e. ∣H∣=pe.

Such a subgroup is called a Sylow ppp-subgroup. It has maximal prime power order within GGG.

Example: Consider G=S4G=S_4G=S4​, since ∣S4∣=24=23⋅3|S_4|=24=2^3\cdot 3∣S4​∣=24=23⋅3, the Sylow theorem states that there is a subgroup of order 888.
In fact, we can take

H=⟨(12),(34),(13)(24)⟩.H=\langle (12),(34),(13)(24)\rangle. H=⟨(12),(34),(13)(24)⟩.

Example: For G=D5G=D_5G=D5​ is the dihedral group, we have ∣D5∣=10=2⋅5|D_5|=10=2\cdot 5∣D5​∣=10=2⋅5. So there must be subgroups of size 5 and 2. A subgroup generated
by a rotation ⟨ρ2π/5⟩\langle \rho_{2\pi/5}\rangle⟨ρ2π/5​⟩ has order 5. A subgroup generated by any reflection has order 2.

(Corollary of Sylow I) If prime number ppp divides ∣G∣|G|∣G∣, then there exists an element x∈Gx\in Gx∈G with order ppp.

Proof: Using Sylow I, there exists a subgroup H≤GH\leq GH≤G such that ∣H∣=pe|H|=p^e∣H∣=pe. Pick any element y∈Hy\in Hy∈H, one knows ⟨y⟩≤H\langle y\rangle\leq H⟨y⟩≤H and thus the order of yyy is ∣⟨y⟩∣=pk|\langle y\rangle|=p^k∣⟨y⟩∣=pk by Lagrange Theorem. Then x=ypk−1x=y^{p^{k-1}}x=ypk−1 is an element of order ppp.
□\square□

# The Second Sylow Theorem

The first Sylow theorem states that a Sylow ppp-subgroup, a subgroup of maximal size pep^epe dividing ∣G∣|G|∣G∣, exists. In fact, we can say a lot more about what these subgroups look like.

(Sylow II) There are two parts; part one is what is usually referred to as the second Sylow theorem.

  • Given H≤GH\leq GH≤G, where HHH is a Sylow ppp-subgroup, any other Sylow ppp-subgroup H′≤GH'\leq GH′≤G is conjugate to HHH; i.e., there exists g∈Gg\in Gg∈G such that H′=gHg−1H'=gHg^{-1}H′=gHg−1.
  • Given any subgroup K≤GK\leq GK≤G such that ∣K∣=pd|K|=p^d∣K∣=pd, for any Sylow subgroup HHH, there exists g∈Gg\in Gg∈G such that gKg−1≤HgKg^{-1}\leq HgKg−1≤H.
    Notice that ∣K∣|K|∣K∣ does not have to be the maximal prime power, and can have order smaller than ∣H∣|H|∣H∣. Every prime power order subgroup, up to conjugation, sits inside a Sylow subgroup.

Evidently, conjugating a Sylow subgroup will result in a Sylow subgroup (since they have the same size), and Sylow II states that all the Sylow subgroups arise in this way.

Note that the second part is stronger, since ∣K∣|K|∣K∣ can be a prime power smaller than ∣H∣|H|∣H∣, and implies the first part by applying to K=HK = HK=H.
The second part states that given any prime power subgroup K≤GK \leq GK≤G and any Sylow subgroup H≤GH \leq GH≤G, it is possible to conjugate KKK to make it land in HHH.

Since the conjugation of a group is also a group, with the same size, thus the converse of the first part is true.

Example: For D2nD_{2n}D2n​, every subgroup of size 2 is generated by a refection, and Sylow II indicates that all the refections are conjugate.

# The Third Sylow Theorem

The last Sylow theorem indicates the number of these (conjugate) subgroups.

(Sylow III) The number of Sylow ppp-subgroups of GGG divides

m=npe,m=\frac{n}{p^e}, m=pen​,

and is congruent to 1 modulo ppp.

This theorem seems kind of weird, but is actually very useful.

Example: Consider D5D_5D5​ and p=2p = 2p=2. The number of Sylow 222−subgroups is 555, which does divide 10/210/210/2 and is congruent
to 111 modulo 222.

Summary: The first Sylow theorem indicates existence of Sylow subgroups, the second Sylow theorem indicates that all Sylow subgroups are related by conjugation, and the third provides strong (and kind of funky) constraints on the number of such subgroups.

# Applications of the Sylow Theorems

Example: Consider any group GGG of order ∣G∣=15=5⋅3|G|=15=5\cdot 3∣G∣=15=5⋅3, by Sylow III, for p=5p=5p=5, the number of Sylow 555-subgroups divides 3=15/53=15/53=15/5 and is equal to 1 modulo 5. Therefore the only possibility is that the number of Sylow 5-subgroup is 1.
So there is a unique H≤GH \leq GH≤G such that ∣H∣=5|H| = 5∣H∣=5. Since there is only one subgroup of size 5, Sylow II indicates that gHg−1=HgHg^{-1} = HgHg−1=H.
That is, HHH is normal: the Sylow theorems indicate automatically that there is a normal subgroup of size 5.
For p=3p = 3p=3, Sylow III states that the number of Sylow 3−subgroups divides 5 and is 1 mod 3, so it is also 1.
Thus, there exists some unique K⊴GK \unlhd GK⊴G such that ∣K∣=3|K| = 3∣K∣=3.

Moreover, H∩K={1}H \cap K = \{1\}H∩K={1}, since HHH is the cyclic group of order 5 and KKK is the cyclic group of order 3.
Nontrivial elements of HHH have order 5, while elements of KKK have order 3, so they intersect only at the identity.
So the Sylow theorems give, for free, nonintersecting normal subgroups of size 5 and 3 for every single group of size 15. That is quite impressive!

Moreover, we can prove that

  • for h∈H,k∈Kh\in H,k\in Kh∈H,k∈K, hk=khhk=khhk=kh;
  • H×K≅GH\times K\cong GH×K≅G.

Proof: For the first statement, since KKK is normal, thus hkh−1∈Khkh^{-1}\in Khkh−1∈K, and therefore hkh−1k−1∈Khkh^{-1}k^{-1}\in Khkh−1k−1∈K.
Similarly, since HHH is normal, hkh−1k−1∈Hhkh^{-1}k^{-1}\in Hhkh−1k−1∈H. Because H∩K={1}H\cap K=\{1\}H∩K={1}, thus hkh−1k−1=1hkh^{-1}k^{-1}=1hkh−1k−1=1 and hk=khhk=khhk=kh.
The second part follows from the first one, one can verify that (h,k)→hk(h,k)\to hk(h,k)→hk is an isomorphism.
□\square□

Another application of the Sylow theorems is the decomposition of finite abelian groups.
Consider an abelian group GGG such that

∣G∣=p1e1...prer.|G|=p_1^{e_1}...p_r^{e_r}. ∣G∣=p1e1​​...prer​​.

Then we know that we have a Sylow subgroup HiH_iHi​ such that

∣Hi∣=piei.|H_i|=p_i^{e_i}. ∣Hi​∣=piei​​.

Since GGG is abelian, conjugating a group produces the same group, so by Sylow II, these (abelian) subgroups HiH_iHi​ are unique for each prime.

Every finite abelian group GGG is isomorphic to a product of groups of prime power order.

One can verify that G≅H1×...×HrG\cong H_1\times...\times H_rG≅H1​×...×Hr​, where f:(x1,...,xr)↦x1+...+xrf:(x_1,...,x_r)\mapsto x_1+...+x_rf:(x1​,...,xr​)↦x1​+...+xr​ is an isomorphism.

# Proof of Sylow I

Let SSS be the set of subsets of GGG of size pep^epe, and nnn be the order of group GGG. Then by basic combinatorics,

∣S∣=(npe).|S|=\begin{pmatrix} n\\ p^e \end{pmatrix}. ∣S∣=(npe​).

Let GGG act on SSS by left translations:

g:U↦gU.g:U\mapsto gU. g:U↦gU.

Our eventual goal is to fnd a subgroup of GGG of size ppp by looking at stabilizers, as they are always subgroups of GGG.
We find the size of a stabilizer by trying to find an orbit of size mmm (where ∣G∣=m⋅pe|G|=m\cdot p^e∣G∣=m⋅pe), as we then know that the stabilizer will be order pep^epe by orbit-stabilizer theorem.

Lemma: For ∣G∣=n=m⋅pe|G|=n=m\cdot p^e∣G∣=n=m⋅pe such that gcd(m,p)=1gcd(m,p)=1gcd(m,p)=1, one has

∣S∣=(npe)=mmod  p.|S|=\begin{pmatrix} n\\ p^e \end{pmatrix}=m\mod p. ∣S∣=(npe​)=mmodp.

考虑展开(1+x)n(1+x)^n(1+x)n 考察系数？这个就是一个数论问题。

Lemma: Suppose we have a subset U∈SU\in SU∈S, which is a subset of GGG. Also, let HHH be a subgroup of GGG that stabilizes UUU. Then, ∣H∣|H|∣H∣ divides ∣U∣|U|∣U∣.

Proof: Since HHH stabilizes UUU, for any h∈Hh\in Hh∈H, hU=UhU=UhU=U. In other words, for any u∈Uu\in Uu∈U,

Hu⊆U.Hu\subseteq U. Hu⊆U.

This implies that the right cosets partition UUU. Since the cosets have the same size, we know that ∣H∣|H|∣H∣ divides ∣U∣|U|∣U∣.
□\square□

We know that the orbits partition SSS, so

∣S∣=∣O1∣+...+∣Or∣.|S|=|O_1|+...+|O_r|. ∣S∣=∣O1​∣+...+∣Or​∣.

Since ppp does not divide LHS, there must exist an orbit θ\thetaθ such that

gcd(p,∣θ∣)=1.gcd(p,|\theta|)=1. gcd(p,∣θ∣)=1.

注意，θ\thetaθ 是SSS 的子集，是一个 “大小为pep^epe 的GGG 的子集的集合”。
Let the size of θ\thetaθ be kkk.
Now, consider some element U∈θU\in \thetaU∈θ. By the orbit-stabilizer theorem, we also know that

∣G∣=∣θ∣⋅∣Stab(U)∣.|G|=|\theta|\cdot |Stab(U)|. ∣G∣=∣θ∣⋅∣Stab(U)∣.

And so pem=k∣Stab(U)∣p^em=k|Stab(U)|pem=k∣Stab(U)∣ and pe∣∣Stab(U)∣p^e\mid |Stab(U)|pe∣∣Stab(U)∣ since gcd(k,p)=1gcd(k,p)=1gcd(k,p)=1. By the second lemma, ∣Stab(U)∣|Stab(U)|∣Stab(U)∣ divides ∣U∣=pe|U|=p^e∣U∣=pe, thus ∣Stab(U)∣=pe|Stab(U)|=p^e∣Stab(U)∣=pe.
□\square□

# Proof of Sylow II

Fix HHH to be a Sylow subgroup. Consider a set X=G/HX = G/HX=G/H, the left cosets of HHH. The index of HHH is the same as ∣X∣|X|∣X∣, so ∣X∣=∣G∣/∣H∣=mpepe=m|X| = |G|/|H|=\frac{mp^e}{p^e}=m∣X∣=∣G∣/∣H∣=pempe​=m.

Let KKK be the subgroup we want to show is a subgroup of HHH up to conjugation, where ∣K∣=pf|K| = p^f∣K∣=pf.
We will look at how KKK acts on XXX by left translation, the mapping:

k(aH)↦(ka)H.k(aH)\mapsto (ka)H. k(aH)↦(ka)H.

We decompose into orbits, ∣X∣=∣O1∣+...+∣Or∣|X|=|O_1|+...+|O_r|∣X∣=∣O1​∣+...+∣Or​∣. Note that these orbits are with respect to the action of KKK, not the action of GGG, as that would be transitive and we’d only have one orbit. That is, Oi=K⋅xiO_i=K\cdot x_iOi​=K⋅xi​ for xi∈Hx_i\in Hxi​∈H.
We have that ∣Oi∣|O_i|∣Oi​∣ divides ∣K∣=pf|K|=p^f∣K∣=pf （因为 orbit-stabilizer theorem， ∣K∣=∣O∣⋅∣Stab∣|K|=|O|\cdot |Stab|∣K∣=∣O∣⋅∣Stab∣）, but ppp does not divide mmm. Thus this orbit decomposition can only work if some orbit OOO has size 1.
In other words, there exists some coset aHaHaH that is fixed by all k∈Kk \in Kk∈K. Then,

kaH=aH  ⟹  a−1ka∈H,kaH=aH\implies a^{-1}ka\in H, kaH=aH⟹a−1ka∈H,

for all k∈Kk\in Kk∈K. Thus a−1Ka≤Ha^{-1}Ka\leq Ha−1Ka≤H.
□\square□
