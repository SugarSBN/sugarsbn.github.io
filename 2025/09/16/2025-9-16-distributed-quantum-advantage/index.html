# LCL Problem in the black-white formalism

我们首先考虑一类问题，被称为 "locally checkable labeling" (LCL) 问题。
我们考虑一种它的特殊形式，即 LCL problem in the black-white formalism。

考虑一个二分图G=(V,E)G=(V,E)G=(V,E)，其中V=W∪BV=W\cup BV=W∪B 是二分图的两个部分，第一部分被称为 White nodes，第二部分是 Black nodes。

  • Σin,Σout\Sigma_{in},\Sigma_{out}Σin​,Σout​ 是两个 finite sets，包含了 labels；
  • Cw,Cb\mathcal{C}_w,\mathcal{C}_bCw​,Cb​ 是两个 sets of multisets，其中每个 multisets 中的元素都是来自于Σin×Σout\Sigma_{in}\times \Sigma_{out}Σin​×Σout​。

其中Cw,Cb\mathcal{C}_w,\mathcal{C}_bCw​,Cb​ 理解为某种 constraints，即合法的 labeling 方式。形式化地，给定一个把所有 edge 都 labeling 的 assignmenti:E→Σini:E\to \Sigma_{in}i:E→Σin​ 作为 input，一个 assignment o:E→Σouto:E\to \Sigma_{out}o:E→Σout​ 被称为是一个 solution to Π\PiΠ 如果它满足 while and black constraints，即：

# Port-numbering model

一个 port-numbered network 被形式化地定义为一个 triple N=(V,P,p)N=(V,P,p)N=(V,P,p)，其中

  • VVV 是 nodes 的集合；
  • PPP 是 ports 的集合，每个 node vvv 的 port 是：(v,1),(v,2),...,(v,dv)(v,1),(v,2),...,(v,d_v)(v,1),(v,2),...,(v,dv​)，其中dvd_vdv​ 是vvv 的 degree；
  • p:P→Pp:P\to Pp:P→P 是一个函数 specifying connections between ports，满足p(p(x))=xp(p(x))=xp(p(x))=x。

显然，在一个 port-numbered network 中，两个 nodes u,vu,vu,v 相连当且仅当存在u,vu,vu,v 的 ports xu,xvx_u,x_vxu​,xv​ 使得p(xu)=xvp(x_u)=x_vp(xu​)=xv​。

我们考虑这样一个 port-numbered network 上的同步分布式算法，其中每个 node 对应一个分布式结点。此外，结合之前的 LCL 问题，给定 input i:E→Σini:E\to \Sigma_{in}i:E→Σin​，我们希望最后每个 white 结点都可以 locally 地计算出它每个 port 对应的边的 output label，即o:E→Σouto:E\to \Sigma_{out}o:E→Σout​。

LOCAL model 就是在此基础上，给每个结点一个唯一的 ID：id(v)∈[nc]id(v)\in [n^c]id(v)∈[nc]，其中ccc 是一个常数。然后如果允许 randomization，那么我们默认需要成功概率至少在1−1/n1-1/n1−1/n。quantum-LOCAL model 是类似的，除了每个 node 可以对一个 local state 作任意操作，并且可以通过 quantum channel 传递信息。
