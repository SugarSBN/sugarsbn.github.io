我们从一些补充的谱分析（spectral analysis）开始。

首先，我们定义一个 unital algebra AAA 中的一个 element aaa 的谱 (spectrum) 为

σ(a)={λ∈C:λ1−a is not invertible}.\sigma(a)=\{\lambda\in\mathbb{C}: \lambda 1 - a \text{ is not invertible}\}. σ(a)={λ∈C:λ1−a is not invertible}.

相应的，谱半径 (spectral radius) 定义为

ρ(a)=sup⁡{∣λ∣:λ∈σ(a)}.\rho(a)=\sup\{|\lambda|:\lambda\in\sigma(a)\}. ρ(a)=sup{∣λ∣:λ∈σ(a)}.

那么首先，spectrum 保持多项式运算：

Theorem: 对于一个 unital algebra AAA 中的 element aaa，和任意复系数多项式p(z)p(z)p(z)，如果σ(a)\sigma(a)σ(a) 不为空，那么

σ(p(a))=p(σ(a))={p(λ):λ∈σ(a)}.\sigma(p(a))=p(\sigma(a))=\{p(\lambda):\lambda\in\sigma(a)\}. σ(p(a))=p(σ(a))={p(λ):λ∈σ(a)}.

Proof：ppp 是 constant 多项式时是显然的。我们假设ppp 不是常数多项式，那么对于任意一个复数μ∈C\mu\in\mathbb{C}μ∈C，都存在λ0,λ1,...,λn∈C\lambda_0,\lambda_1,...,\lambda_n\in\mathbb{C}λ0​,λ1​,...,λn​∈C 使得

p(z)−μ=λ0(z−λ1)(z−λ2)...(z−λn),p(z)-\mu=\lambda_0(z-\lambda_1)(z-\lambda_2)...(z-\lambda_n), p(z)−μ=λ0​(z−λ1​)(z−λ2​)...(z−λn​),

p(a)−μ1=λ0(a−λ11)(a−λ21)...(a−λn1).p(a)-\mu 1=\lambda_0(a-\lambda_1 1)(a-\lambda_2 1)...(a-\lambda_n 1). p(a)−μ1=λ0​(a−λ1​1)(a−λ2​1)...(a−λn​1).

注意到p(a)−μ1p(a)-\mu 1p(a)−μ1 可逆当且仅当a−λ11,a−λ21,...,a−λn1a-\lambda_11,a-\lambda_21,...,a-\lambda_n1a−λ1​1,a−λ2​1,...,a−λn​1 都可逆。

换句话说，μ∈σ(p(a))\mu\in \sigma(p(a))μ∈σ(p(a)) 当且仅当存在一个λi\lambda_iλi​ 使得a−λi1a-\lambda_i1a−λi​1 不可逆，即λi∈σ(a)\lambda_i\in\sigma(a)λi​∈σ(a)。注意到恒有p(λi)−μ=0p(\lambda_i)-\mu=0p(λi​)−μ=0，所以μ=p(λi)∈p(σ(a))\mu=p(\lambda_i)\in p(\sigma(a))μ=p(λi​)∈p(σ(a))。

□\square□

Theorem: 对于一个 unital Banach algebra AAA，如果一个 element a∈Aa\in Aa∈A 满足∥a∥<1\|a\|<1∥a∥<1，那么1−a1-a1−a 可逆且

(1−a)−1=∑n=0∞an.(1-a)^{-1}=\sum_{n=0}^{\infty}a^n. (1−a)−1=n=0∑∞​an.

Proof: 注意到

∑n=0∞∥an∥≤∑n=0∞∥a∥n=11−∥a∥<∞,\sum_{n=0}^\infty \|a^n\|\leq \sum_{n=0}^\infty \|a\|^n=\frac{1}{1-\|a\|}<\infty, n=0∑∞​∥an∥≤n=0∑∞​∥a∥n=1−∥a∥1​<∞,

因此∑n=0∞an\sum_{n=0}^\infty a^n∑n=0∞​an 收敛（Banach 空间），不妨设收敛到b∈Ab\in Ab∈A。此外，对于任意nnn，

(1−a)(1+a+a2+...+an)=1−an+1.(1-a)(1+a+a^2+...+a^n)=1-a^{n+1}. (1−a)(1+a+a2+...+an)=1−an+1.

一方面，因为乘法是连续的，所以

lim⁡n→∞(1−a)(1+a+a2+...+an)=(1−a)b.\lim_{n\to\infty}(1-a)(1+a+a^2+...+a^n)=(1-a)b. n→∞lim​(1−a)(1+a+a2+...+an)=(1−a)b.

另一方面，

lim⁡n→∞∥an∥≤lim⁡n→∞∥a∥n=0,\lim_{n\to\infty}\|a^n\|\leq \lim_{n\to\infty}\|a\|^n=0, n→∞lim​∥an∥≤n→∞lim​∥a∥n=0,

因此

(1−a)(1+a+a2+...+an)=lim⁡n→∞(1−an+1)=1.(1-a)(1+a+a^2+...+a^n)=\lim_{n\to\infty}(1-a^{n+1})=1. (1−a)(1+a+a2+...+an)=n→∞lim​(1−an+1)=1.

所以b(1−a)=1b(1-a)=1b(1−a)=1，同理可以证明(1−a)b=1(1-a)b=1(1−a)b=1。

□\square□

接下来我们还会提供几个定理，但证明需要更多额外内容。

Theorem (Gelfrand): 对于一个 unital Banach algebra AAA 中的任意 element aaa，它的 spectrum σ(a)\sigma(a)σ(a) 非空。

Theorem (Beurling): 对于一个 unital Banach algebra AAA 中的任意 element aaa，它的 spectral radius ρ(a)\rho(a)ρ(a) 满足

ρ(a)=lim⁡n→∞∥an∥1/n=inf⁡n≥1∥an∥1/n.\rho(a)=\lim_{n\to\infty}\|a^n\|^{1/n}=\inf_{n\geq 1}\|a^n\|^{1/n}. ρ(a)=n→∞lim​∥an∥1/n=n≥1inf​∥an∥1/n.

Theorem: 令BBB 是一个 closed subalgebra of a unital Banach algebra AAA 且包含单位元 1，那么

  • Inv(B)Inv(B)Inv(B) 是B∩Inv(A)B\cap Inv(A)B∩Inv(A) 的一个既开又闭集合，其中Inv(B)Inv(B)Inv(B) 表示在BBB 中可逆的元素集合。
  • 对于任意b∈Bb\in Bb∈B，σA(b)⊆σB(b)\sigma_A(b)\subseteq \sigma_B(b)σA​(b)⊆σB​(b)，并且∂σB(b)⊆∂σA(b)\partial\sigma_B(b)\subseteq \partial \sigma_A(b)∂σB​(b)⊆∂σA​(b)。

接下来进入算子代数部分。

Theorem：对于一个C∗C^*C∗-algebra AAA 中的 self-adjoint element aaa，它的 spectral radius ρ(a)\rho(a)ρ(a) 等于∥a∥\|a\|∥a∥。

Proof: 我们知道∥a2∥=∥a∗a∥=∥a∥2\|a^2\|=\|a^*a\|=\|a\|^2∥a2∥=∥a∗a∥=∥a∥2，因此∥a2n∥=∥a∥2n\|a^{2^n}\|=\|a\|^{2^n}∥a2n∥=∥a∥2n，所以

ρ(a)=lim⁡n→∞∥an∥1/n≥lim⁡n→∞∥a2n∥1/2n=∥a∥.\rho(a)=\lim_{n\to\infty}\|a^n\|^{1/n}\geq \lim_{n\to\infty}\|a^{2^n}\|^{1/2^n}=\|a\|. ρ(a)=n→∞lim​∥an∥1/n≥n→∞lim​∥a2n∥1/2n=∥a∥.

□\square□

Corollary: 一个∗*∗-algebra 上至多有一个 norm 使得它成为一个C∗C^*C∗-algebra。

Proof：根据上一个 theorem，如果有一个 norm 成为了C∗C^*C∗-algebra，那么对于任意 element aaa，

∥a∥2=∥a∗a∥=ρ(a∗a)=sup⁡λ∈σ(a∗a)∣λ∣.\|a\|^2=\|a^*a\|=\rho(a^*a)=\sup_{\lambda\in\sigma(a^*a)}|\lambda|. ∥a∥2=∥a∗a∥=ρ(a∗a)=λ∈σ(a∗a)sup​∣λ∣.

因此不同的 norm 应该有相同的取值。

□\square□

Lemma: 令AAA 是一个 Banach ∗*∗-algebra，如果有∥a∥2≤∥a∗a∥\|a\|^2\leq \|a^*a\|∥a∥2≤∥a∗a∥，那么AAA 是一个C∗C^*C∗-algebra。

Proof: 注意到∥a∥2≤∥a∗a∥≤∥a∥∥a∗∥\|a\|^2\leq \|a^*a\|\leq \|a\|\|a^*\|∥a∥2≤∥a∗a∥≤∥a∥∥a∗∥，因此∥a∥≤∥a∗∥\|a\|\leq \|a^*\|∥a∥≤∥a∗∥。同理∥a∗∥2≤∥aa∗∥≤∥a∥∥a∗∥\|a^*\|^2\leq \|aa^*\|\leq \|a\|\|a^*\|∥a∗∥2≤∥aa∗∥≤∥a∥∥a∗∥，所以∥a∥=∥a∗∥\|a\|=\|a^*\|∥a∥=∥a∗∥。因此∥a∗a∥=∥a∥2\|a^*a\|=\|a\|^2∥a∗a∥=∥a∥2。
□\square□

Theorem: 对于一个C∗C^*C∗-algebra AAA 中的任意 element aaa，如果aaa 是 self-adjoint 的，那么σ(a)⊆R\sigma(a)\subseteq\mathbb{R}σ(a)⊆R。

Proof：对于一个λ∈σ(a)\lambda\in\sigma(a)λ∈σ(a)，假设λ=α+iβ\lambda=\alpha+i\betaλ=α+iβ，我们证明β=0\beta=0β=0。注意到

(a−λ1)∗(a−λ1)=(a−α1)2+β21.(a-\lambda 1)^*(a-\lambda 1)=(a-\alpha 1)^2+\beta^21. (a−λ1)∗(a−λ1)=(a−α1)2+β21.

那么根据 spectrum 保持多项式（之前的 theorem），我们有

σ((a−α1)2+β21)=(σ(a)−α)2+β2.\sigma((a-\alpha 1)^2+\beta^21)=(\sigma(a)-\alpha)^2+\beta^2. σ((a−α1)2+β21)=(σ(a)−α)2+β2.

如果β≠0\beta\neq 0β=0，那么σ((a−α1)2+β21)>0\sigma((a-\alpha 1)^2+\beta^21)>0σ((a−α1)2+β21)>0，所以(a−λ1)∗(a−λ1)(a-\lambda 1)^*(a-\lambda 1)(a−λ1)∗(a−λ1) 可逆（因为 0 不在它的 spectrum 里）。

因此(a−λ1)(a-\lambda 1)(a−λ1) 也可逆，这与λ∈σ(a)\lambda\in\sigma(a)λ∈σ(a) 矛盾。因此β=0\beta=0β=0。

补充一下：如果b∗bb^*bb∗b 可逆那么bbb 也可逆，因为σ(b∗b)={∣λ∣2:λ∈σ(b)}\sigma(b^*b)=\{|\lambda|^2:\lambda\in\sigma(b)\}σ(b∗b)={∣λ∣2:λ∈σ(b)}。
□\square□

令BBB 是 unital C∗C^*C∗-algebra AAA 的一个C∗C^*C∗-subalgebra 且包含单位元 1，那么对于任意b∈Bb\in Bb∈B，它在BBB 中的 spectrum 等于它在AAA 中的 spectrum。

Proof：我们证明bbb 在BBB 中的可逆性等价于在AAA 中的可逆性。只有一个方向是 non-trivial 的：即bbb 在AAA 中可逆，那么bbb 在BBB 中也可逆。

如果存在a∈Aa\in Aa∈A 使得ab=ba=1ab=ba=1ab=ba=1，那么a∗b∗=b∗a∗=1a^*b^*=b^*a^*=1a∗b∗=b∗a∗=1。

□\square□
