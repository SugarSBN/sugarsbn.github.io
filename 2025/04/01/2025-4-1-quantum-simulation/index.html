# 背景设定和定性分析

考虑薛定谔方程

i∂∂t∣ψ(t)⟩=H(t)∣ψ(t)⟩,i\frac{\partial}{\partial t}|\psi(t)\rangle = H(t)|\psi(t)\rangle, i∂t∂​∣ψ(t)⟩=H(t)∣ψ(t)⟩,

其中H(t)∈CN×NH(t)\in \mathbb{C}^{N\times N}H(t)∈CN×N 是一个 Hermitian matrix，其中N=2nN=2^nN=2n。
我们希望找到一个 gate complexity poly(n,t,1/ϵ)poly(n,t,1/\epsilon)poly(n,t,1/ϵ) 的量子算法，来模拟∣ψ(0)⟩↦∣ψ(t)⟩|\psi(0)\rangle\mapsto |\psi(t)\rangle∣ψ(0)⟩↦∣ψ(t)⟩。
（其中我们允许算法输出和∣ψ(t)⟩|\psi(t)\rangle∣ψ(t)⟩ 的 norm 差≤ϵ\leq \epsilon≤ϵ）
如果这个算法能找到，我们说H(t)H(t)H(t) 是可以被 efficiently simulate 的。

下面我们考虑定时 Hamiltonian H(t)H(t)H(t)，即H(t)=HH(t)=HH(t)=H 是一个常数矩阵。

  1. (Local) 如果HHH acts on O(1)O(1)O(1) qubits，那么HHH 是可以被 efficiently simulate 的。
  2. (Scalability) 如果HHH 是可以被 efficiently simulate 的，那么c=poly(n)c=poly(n)c=poly(n) 时c⋅Hc\cdot Hc⋅H 也可以被 efficiently simulate 的。
  3. (Unitary) 如果HHH 是可以被 efficiently simulate 的，那么UHU†UHU^\daggerUHU† 也可以被 efficiently simulate 的，其中UUU 是 unitary。
  4. (Addition) 如果H1H_1H1​ 和H2H_2H2​ 是可以被 efficiently simulate 的，那么H1+H2H_1+H_2H1​+H2​ 也可以被 efficiently simulate 的。(注意H1,H2H_1,H_2H1​,H2​ 可能不交换)
  5. (Commutator) 如果H1H_1H1​ 和H2H_2H2​ 是可以被 efficiently simulate 的，那么i[H1,H2]i[H_1,H_2]i[H1​,H2​] 也可以被 efficiently simulate 的。
  6. (Diagonal) 如果HHH 是对角的，那么HHH 是可以被 efficiently simulate 的。
  7. (Sparse) 如果HHH 是O(1)O(1)O(1)-sparse 的，那么HHH 是可以被 efficiently simulate 的。这里会有一些 matrix access oracle，关于 oracle 的 query complexity 也是 polynomial 的。

Example: 对于一个粒子的 Hamiltonian H=p22m+V(x)H=\frac{p^2}{2m}+V(x)H=2mp2​+V(x)，其中p2=−d2dx2p^2=-\frac{d^2}{dx^2}p2=−dx2d2​ 是 square of the momentum operator，p22m\frac{p^2}{2m}2mp2​ 是动能部分，V(x)V(x)V(x) 是势能部分。势能部分是对角的，P2P^2P2 可以在 Fourier transformation（unitary）下是对角的，因此HHH 是可以被 efficiently simulate 的。

考虑H=A+BH=A+BH=A+B，而且∥H1∥>>∥H2∥\|H_1\|>>\|H_2\|∥H1​∥>>∥H2​∥ 时。考虑

∣ψ1(t)⟩≜eiH1t∣ψ(t)⟩,|\psi_1(t)\rangle\triangleq e^{iH_1t}|\psi(t)\rangle, ∣ψ1​(t)⟩≜eiH1​t∣ψ(t)⟩,
