# 和谐

定义：交融共生，包容矛盾，丰富充实，心灵体验。和谐是交响乐而不是独奏

联想：1、圣埃克絮佩里：飞行的风险、黑暗的神秘、星空的灿烂、风雨的无情。独特体验与文学作品交融，人格价值与文学价值结合，成为和谐的大写的人。

​ 2、《要塞》：“你同意为之而死的东西，就是你能够为之而生的东西”

​ 3、索尔仁尼琴 “一个世界不能只有一种声音”

​ 4、卡尔维诺《分成两半的子爵》善恶冲突，人性分裂到和谐统一

​ 5、《树上的男爵》通向完整的道路

​ 6、格里尔帕策 “我热爱人类，特别是他们聚在一起，暂时忘记了各人的私立，融合为 一个整体时，因为神性就存在于这样一个整体上”

​ 7、里尔克《给青年诗人的信》“也许你自身内就负有可能性：去组织，去形成一种特别幸福与纯洁的生活方式；你要向那方面修养 —— 但是，无论什么来到，你都要以广大的信任领受；如果它是从你意志里、从任何一种内身的窘困里产生的，那么你要好好负担着它，什么也不要憎恶 “与自己和解，与过去和解

​ 8、《德米安》“我们来自同一个深渊，可以理解彼此，然而能解读自己的人只有自己”

联想：1、《德米安》“觉醒的人只有一项义务，找到自我，然后在心中坚守其一生”

6、小仲马 “对于有接触智慧的人来说，热情应之应该是一个动力，正像风对于大海一样，风卷起海浪，使大海发怒，气势雄伟，然后风消失了，大海还是大海” 八风不动

7、叔本华 “孤独时，人须委身于自己”“一个人内在所具备的愈多，求之于他人的愈少，他人能给自己的也愈少”

8、鲁迅 “我们极易变成奴隶，而且变了之后还万分欢喜”

9、尼采的酒神精神和日神精神，走向自我和走向世界。

10、《圣经》“赐我沉静，接受我必须接受的；赐我勇气，改变我可以改变的；赐我智慧，分辨这二者的区别。“

3、里尔克《布里格随笔》“被爱意味着被消耗，被染成灰烬，爱则意味着永不枯息的明灯放射光芒，被爱转瞬即逝，爱则永存不灭”

4、陀氏 “在这个世界上，人只有通过痛苦才能真正爱”

5、雨果 “人生是花，而爱是花蜜”

6、但丁 “爱是美德的种子”

7、罗曼罗兰 “爱是生命的火焰，没有它，一切变成黑夜”

8、狄金森 “如果我能使一颗心免于破碎，我将没有虚度此生”

# 经验（参考回眸）

联想：1、帕特里克 “我只有一盏指路明灯，那就是经验之灯”

​ 2、雅斯贝尔斯 “从来只有一条道路，那就是穿过历史的道路”

3、朗费罗 “不要老叹息过去，它是不再回来的，要明智地改善现在。要以不忧不惧的坚决意志投入扑朔迷离的未来”

4、梅里迪斯 “过去是我们临终的母亲，并不是已经死亡的事物。我们的未来不断使她出现在我们心灵中”

# 自由

联想：1、黑格尔 “人们往往把任性也叫做自由，但是任性只是非理性的自由”

# 兴趣

联想：1、黑格尔 “一个深广的心灵总是把兴趣的领域推广到无数事物上去”

​ 2、勃洛克 “我偶然在一把小刀上发现了异国的一粒微尘 —— 世界顿时又变得奇异万 状，缭绕着五彩缤纷的氤氲”
