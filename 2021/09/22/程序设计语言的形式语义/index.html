# 操作语义

  • IMP 是一种带 while 的小型程序设计语言，其程序在运行时要依靠一连串的显式命令进行状态转换，因此 IMP 被称为是一种命令式语言。形式上 IMP 由一组规则来描述，规定了表达式如何计算和命令如何执行。规则也是证明命令间等价的基本工具。

# 算术表达式的求值

  • 状态集合Σ\SigmaΣ 由存储单元到数集的函数σ:Loc→N\sigma:Loc\rightarrow Nσ:Loc→N 组成。于是σ(X)\sigma(X)σ(X) 是状态σ\sigmaσ 下存储单元 X 的值或内容。下面讨论在状态σ∈Σ\sigma\in\Sigmaσ∈Σ 下算术表达式aaa 的求值过程。

  • 用序偶（有序对）⟨a,σ⟩\lang a,\sigma\rang⟨a,σ⟩ 来表示在状态σ\sigmaσ 下表达式aaa 等待求值。并把这个序偶与数集间的求值关系定义为：

    ⟨a,σ⟩→n\lang a,\sigma\rang\rightarrow n ⟨a,σ⟩→n

    它表示在状态σ\sigmaσ 下表达式aaa 的求值结果为nnn。而序偶⟨a,σ⟩\lang a,\sigma\rang⟨a,σ⟩ 为算术表达式的格局。

  • 考察算术表达式a0+a1a_0+a_1a0​+a1​ 的求值过程，其顺序为：

    • 求值a0a_0a0​ 得到结果n0n_0n0​
    • 求值a1a_1a1​ 得到结果n1n_1n1​
    • 将n0n_0n0​ 与n1n_1n1​ 相加得到a0+a1a_0+a_1a0​+a1​ 的求值结果。

    整个过程通过分别求值加式和被加式然后相加得到求值结果，这种规范被称为语法制导下的规范。

语法制导下算术表达式的求值关系按以下规则指定：

⟨a0,σ⟩→n0⟨a1,σ⟩→n1⟨a0+a1,σ⟩→n,n=n0+n1\frac{\lang a_0,\sigma\rang\rightarrow n_0\quad\lang a_1,\sigma\rang\rightarrow n_1}{\lang a_0+a_1,\sigma\rang\rightarrow n},n=n_0+n_1 ⟨a0​+a1​,σ⟩→n⟨a0​,σ⟩→n0​⟨a1​,σ⟩→n1​​,n=n0​+n1​

⟨a0,σ⟩→n0⟨a1,σ⟩→n1⟨a0×a1,σ⟩→n,n=n0×n1\frac{\lang a_0,\sigma\rang\rightarrow n_0\quad\lang a_1,\sigma\rang\rightarrow n_1}{\lang a_0\times a_1,\sigma\rang\rightarrow n},n=n_0\times n_1 ⟨a0​×a1​,σ⟩→n⟨a0​,σ⟩→n0​⟨a1​,σ⟩→n1​​,n=n0​×n1​

    这就是一棵求值树，也称推导树（简称推导）。推导树由规则的实例构成，每个规则实例的前提正好是上一层实例的结论，公理位于最顶层。最底层的实例的结论为整个推导的结论。如果某个推导存在结论，则说该结论是从规则可精确推导的。
