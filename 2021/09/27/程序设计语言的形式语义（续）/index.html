# IMP 的指称语义

# 目的

我们可以用状态集合上的部分函数来表示命令的指称，并称这种新的语义描述风格为指称语义。

以 IMP 语言为例：

  • 算术表达式a∈Aexpa\in Aexpa∈Aexp 的指称函数A[a]:Σ→N\mathscr{A}[a]:\Sigma\rightarrow NA[a]:Σ→N
  • 布尔表达式b∈Bexpb\in Bexpb∈Bexp 的指称函数B[b]:Σ→T\mathscr{B}[b]:\Sigma\rightarrow TB[b]:Σ→T
  • 命令c∈Comc\in Comc∈Com 指称部分函数C[c]:Σ→Σ\mathscr{C}[c]:\Sigma\rightarrow\SigmaC[c]:Σ→Σ

注：方括号应该都为 “空心方括号”，但因为我打不出来 emm… 特此说明，后文也是。

空心方括号是指称语义的一个常用符号。A\mathscr{A}A 实际上是类型为Aexp→(Σ→N)Aexp\rightarrow(\Sigma\rightarrow N)Aexp→(Σ→N) 的算术表达式的一个函数。

# 完全偏序与连续函数

我们引入更抽象的概念 —— 完全偏序和连续函数来定义递归，两者都是指称语义的标准工具。

# Knaster-Tarski 定理

  • 对一组规则实例 R，有

    IR=⋂{Q∣Q对R封闭}I_R=\bigcap\{Q|Q对R封闭\} IR​=⋂{Q∣Q对R封闭}

    命题等价为：

    fix(R^)=⋂{Q∣R^(Q)⊆Q}fix(\hat{R})=\bigcap\{Q|\hat{R}(Q)\subseteq Q\} fix(R^)=⋂{Q∣R^(Q)⊆Q}

    即算子R^\hat{R}R^ 的最小不动点可以用其前缀不动点的交集来刻画。这其实是克纳斯塔 - 塔尔斯基定理的特殊情况。

  • 定义，对偏序(P,⊑)(P,\sqsubseteq)(P,⊑) 和子集X⊆PX\subseteq PX⊆P，称 p 是 X 的下界，当且仅当

    ∀q∈X,p⊑q\forall q\in X,p\sqsubseteq q ∀q∈X,p⊑q

    同样可以类似定义 X 的最大下界（glb）。

    如果一个偏序的子集 X 有最大下界，我们把它记为⊓X\sqcap X⊓X。最小上界又称为上确界（sups），最大下界又称为下确界（infs）。

  • 定义，如果一个偏序的任意子集都有最大下界，则称这个偏序为完全格。事实上，完全格的任意子集也必有最小上界。而且若(L,⊑)(L,\sqsubseteq)(L,⊑) 是完全格，对应的逆关系(L,⊒)(L,\sqsupseteq)(L,⊒) 也是完全格。

  • 克纳斯塔 - 塔尔斯基最小不动点定理，令(L,⊑)(L,\sqsubseteq)(L,⊑) 是完全格，f:L→Lf:L\rightarrow Lf:L→L 是单调函数，定义

    m=⊓{x∈L∣f(x)⊑x}m=\sqcap\{x\in L|f(x)\sqsubseteq x\} m=⊓{x∈L∣f(x)⊑x}

    则 m 是 f 的不动点和 f 的最小前缀不动点。同理，定义

    M=⨆{x∈L∣x⊑f(x)}M=\bigsqcup\{x\in L|x\sqsubseteq f(x)\} M=⨆{x∈L∣x⊑f(x)}

    则 M 是 f 的不动点和 f 的最大后缀不动点。

    证明，以最小前缀不动点为例：

    首先，m 是前缀不动点集合的下界，所以其为最小前缀不动点。

    其次，因为f(m)⊑mf(m)\sqsubseteq mf(m)⊑m，由单调性，有f(f(m))⊑f(m)f(f(m))\sqsubseteq f(m)f(f(m))⊑f(m)，所以f(m)f(m)f(m) 也是集合{x∈L∣f(x)⊑x}\{x\in L|f(x)\sqsubseteq x\}{x∈L∣f(x)⊑x} 的元素。

    又因为 m 是该集合的下界，所以也有m⊑f(m)m\sqsubseteq f(m)m⊑f(m)，所以有m=f(m)m=f(m)m=f(m)。即为不动点。
