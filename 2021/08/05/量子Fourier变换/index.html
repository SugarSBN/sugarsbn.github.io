# 问题的转化：将因子分解转化为求阶

算法过程：

  • 若 N 是偶数，返回因子 2。

  • 验证 N 是否为N=abN=a^bN=ab（利用上述经典算法）若是，返回 a。

  • 随机地在 1 到 N-1 范围内选择一个 x，若gcd(x,N)>1gcd(x,N)>1gcd(x,N)>1，则返回因子gcd(x,N)gcd(x,N)gcd(x,N)。

  • 否则利用求阶子程序，求出 x 模 N 的阶 r。

  • 若 r 是偶数且xr/2≠−1(modN)x^{r/2}\neq -1(modN)xr/2=−1(modN)，则计算gcd(xr/2−1,N),gcd(xr/2+1,N)gcd(x^{r/2}-1,N),gcd(x^{r/2}+1,N)gcd(xr/2−1,N),gcd(xr/2+1,N)，并且测试其中哪一个是 N 的不平凡因子。

  • 若 r 是偶数且gcd(xr/2−1,N),gcd(xr/2+1,N)gcd(x^{r/2}-1,N),gcd(x^{r/2}+1,N)gcd(xr/2−1,N),gcd(xr/2+1,N) 中有一个为 N 的不平凡因子，则返回该因子，否则算法失败。
