    meaning “if in state q1q_1q1​, you see a symbol aaa on the work tape, then replace it with bbb, move in the direction DDD, goto state q2q_2q2​ and print ccc to the printer”. Every time it prints a symbol, the head of the printer moves right. if ccc is absent, then nothing happens to printer.

  • There is start state, but no accept or reject states. Entering a configuration from which there is no transition causes the enumerator to halt.

  • The language enumerated by an enumerator is the set of strings that it prints on the printer.

Example:

Theorem

A language is Turing-recognizable if and only if some enumerator enumerates it.

Proof: First we show that if we have an enumerator EEE that enumerates a language AAA, a TM MMM recognizes it.

MMM = "On input www:

  1. Run EEE. Every time that EEE outputs a string, compare it with www.
  2. If www ever appears in the output of EEE, accept."

Now we do the other direction

EEE = "Ignore the input

  1. Repeat the following for i=1,2,3,...i=1,2,3,...i=1,2,3,...
  2. Run MMM for iii steps on each input s1,s2,...,sis_1,s_2,...,s_is1​,s2​,...,si​
  3. If any computations accept, print out the correspoinding sjs_jsj​."

where s1,s2,s3,...s_1,s_2,s_3,...s1​,s2​,s3​,... is a list of all possible strings in Σ∗\Sigma^*Σ∗.

In fact, if MMM accepts some string www, then it will appear on the printer infinite times.

Church-Turing thesis: Intuitive notion of algorithms = algorithms defined by λ\lambdaλ calculus = algorithms defined by Turing machines.

EDFA={⟨A⟩∣AE_{DFA}=\{\langle A\rangle|AEDFA​={⟨A⟩∣A is a DFA and L(A)=∅}L(A)=\emptyset\}L(A)=∅} is decidable.

TTT = "On input ⟨A⟩\langle A\rangle⟨A⟩ where AAA is a DFA:

  1. Mark the start state of AAA.
  2. Repeat until no new states get marked:
  3. Mark any state that has a transition coming into it from any state that is already marked.

EQDFA={⟨A,B⟩∣AEQ_{DFA}=\{\langle A,B\rangle|AEQDFA​={⟨A,B⟩∣A and BBB are DFAs and L(A)=L(B)}L(A)=L(B)\}L(A)=L(B)} is decidable.

FFF = "On input ⟨A,B⟩\langle A,B\rangle⟨A,B⟩, where AAA and BBB are DFAs:

    L(C)=(L(A)∩∼L(B))∪(∼L(A)∩L(B))L(C)=(L(A)\cap\sim L(B))\cup(\sim L(A)\cap L(B)) L(C)=(L(A)∩∼L(B))∪(∼L(A)∩L(B))

  2. Call ECE_CEC​ to decider whether L(C)L(C)L(C) is empty, if is, then accept. Otherwise reject."

Theorem

ACFG={⟨G,w⟩∣GA_{CFG}=\{\langle G,w\rangle|GACFG​={⟨G,w⟩∣G is a CFG that generates string w}w\}w} is decidable.

Proof:

SSS = "On input ⟨G,w⟩\langle G,w\rangle⟨G,w⟩ where GGG is a CFG and www is a string:

  1. Convert GGG to an equivalent grammar in Chomsky normal form.
  2. List all derivations with 2n−12n-12n−1 steps, where nnn is the length of www, except if n=0n=0n=0, then check if we have the rule S→εS\rightarrow\varepsilonS→ε.
  3. If any of these derivations generate www, accept; if not, reject."

Notice that if a string can be generated by a Chomsky normal form, then the number of derivation steps is at most 2n−12n-12n−1.

ECFG={⟨G⟩∣GE_{CFG}=\{\langle G\rangle|GECFG​={⟨G⟩∣G is a CFG and L(G)=∅}L(G)=\emptyset\}L(G)=∅} is decidable.

  3. Mark any variable AAA where GGG has a rule A→U1U2...UkA\rightarrow U_1U_2...U_kA→U1​U2​...Uk​ and each symbol U1,...,UkU_1,...,U_kU1​,...,Uk​ has already been marked.

Theorem

EQCFG={⟨A,B⟩∣A,BEQ_{CFG}=\{\langle A,B\rangle|A,BEQCFG​={⟨A,B⟩∣A,B are CFGs and L(A)=L(B)}L(A)=L(B)\}L(A)=L(B)} is NOT decidable.

# Mapping reducibility

Definition

A function f:Σ∗→Σ∗f:\Sigma^*\rightarrow\Sigma^*f:Σ∗→Σ∗ is a computable function if some Turing machine MMM, on every input www, halts with just f(w)f(w)f(w) on its tape.

Definition

Language AAA is mapping reducible to language BBB, written A≤mBA\leq_m BA≤m​B, if there is a computable function f:Σ∗→Σ∗f:\Sigma^*\rightarrow\Sigma^*f:Σ∗→Σ∗, where for every www:

w∈A⇔f(w)∈Bw\in A\Leftrightarrow f(w)\in B w∈A⇔f(w)∈B

The function fff is called the reduction of AAA to BBB.

Theorem

If A≤mBA\leq_m BA≤m​B and AAA is undecidable, then BBB is undecidable.

Proof:

If we have a decider for BBB, then for any input www, we can compute f(w)f(w)f(w) and check whether it is in BBB. Then we can decide whether www is in AAA.
