# Isomorphism

Definition(Isomorphism)

Let XXX and YYY be sets. A function f:X→Yf:X\rightarrow Yf:X→Y is called an isomorphism, denoted X→≅YX\stackrel{\cong}{\rightarrow}YX→≅Y, if there exists a function g:Y→Xg:Y\rightarrow Xg:Y→X such that g∘f=idXg\circ f=id_Xg∘f=idX​ and f∘g=idYf\circ g=id_Yf∘g=idY​.

We also say that fff is invertible and we say that ggg is the inverse of fff. If there exists an isomorphism X→≅YX\stackrel{\cong}{\rightarrow} YX→≅Y we say that XXX and YYY are isomorphic sets and may write X≅YX\cong YX≅Y.

# Commutative diagrams

We say this is a diagram of sets if each of A,B,CA,B,CA,B,C is a set and each of f,g,hf,g,hf,g,h is a function. We say this diagram commutes if g∘f=hg\circ f=hg∘f=h.

In this case we refer to it as a commutative triangle of sets. However, the statement here is not precise enough to be a definition.

If g∘f=i∘hg\circ f = i\circ hg∘f=i∘h, then we refer to it as a commutative square of sets.

To say that the diagram commutes would say that every approach gives the same result. (Each approach is a combination of functions).
